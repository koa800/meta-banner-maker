# リストマーケティング・セールス

リストマーケティングの基本概念から、コンテンツ制作、VSL・セミナーセールスの構成フレームワーク、販売導線の仕組み化までを体系的にまとめたナレッジ。広告集客で獲得した見込み客に商品・サービスを届けるまでの全体設計に活用する。

---

## 1. リストマーケティングの基本

### 2つのマーケティング手法の比較

| 項目 | マスマーケティング | リストマーケティング（ダイレクトマーケティング） |
|------|-------------------|--------------------------------------|
| 定義 | 広範囲の見込み客に宣伝する網漁型 | ターゲットを絞って宣伝する一本釣り型 |
| メリット | 多くのユーザーにアプローチ可能 / 短期間で認知拡大 | 少額でパーソナライズ可能 / 詳細な数値計測・改善がしやすい |
| デメリット | 膨大な費用 / パーソナライズ困難 / 効果測定が難しい | 広い顧客層へのアプローチは不得意 / 定期運用しないと興味が移る |
| 代表例 | テレビ、ラジオ、新聞、屋外広告 | 公式LINE、メルマガ |

### 中小企業・個人にリストマーケティングが有効な理由

1. 顕在需要は大手が刈り取っているため、潜在需要に焦点を当てる戦略が取れる
2. ニッチ市場で獲得しやすい
3. 少額でターゲットを絞れるためコスパが良い
4. 信頼を積み上げやすくブランド向上がしやすい

### 広告におけるリストマーケティングの基本フロー

1. **広告集客** -- 見込み客を獲得する
2. **教育** -- 公式LINEやメルマガで価値提供・関係構築を行う
3. **オファー** -- 魅力的な商品をオファーして販売する

## 2. リスク管理

### 主要リスク

| リスク | 概要 |
|--------|------|
| アカウントBAN | 公式LINEが使用不能になり、見込み客への配信が不可能になる |
| 迷惑メール送り | メールが迷惑フォルダに届き開封されなくなる悪循環が発生する |

リスク管理の目的は「復活までのスピードを速くすること」。対策も重要だが、トラブル発生時のリカバリーを早くする設計がより重要。

### アカウントBAN対策（6項目）

1. **公式LINEの分散** -- メインとサブのアカウントを用意し、BAN時にサブから再登録を促す（例: メインシナリオ用LINE + プレゼント受取専用LINE）
2. **メールアドレスの取得** -- メールにはアカウントBANがないため、公式LINE + メールアドレスの両方を取得しておく
3. **同じリンクを何度も送らない** -- セールス時のリンクやリマインドの過多は通報が増える原因になる
4. **グレーな文言を使わない** -- 基本的に使用を避ける
5. **グレーな文言を使う場合は画像で配信** -- NGワードはテキストで判定されている可能性が高い。画像化することでBAN回避の実績あり
6. **エンゲージが低いユーザーには配信しない** -- 通報するのはほぼエンゲージの低いユーザー。特にセールス時は低エンゲージ層を配信対象から除外する

### 一斉送信の注意

- 一斉送信時は通報数が急増するリスクがある
- 急な一斉送信は避け、定期的な配信でリスト運用を継続する

### 迷惑メール対策（2項目）

1. **エンゲージの低い人に配信しない設定にする** -- メール配信ツールで「直近5通未開封の人には送らない」等を設定。商品購入済みの人にセールスメールを送り続けることも避ける
2. **エンゲージの高い人だけの別シナリオを用意する** -- 別フォームでプレゼント配布や別シナリオに誘導するタイミングで、ホットなユーザーのメールアドレスだけを別途取得する

## 3. コンテンツ制作の基礎

### コンテンツの定義

コンテンツ = ユーザーとのコミュニケーションツール。単なる「情報」ではなく、送り手と受け手がコンテンツを通して繋がることで生まれる**体験全体**を指す。

### 良いコンテンツの定義

**良いコンテンツ = 人を動かすコンテンツ**

- どれだけ閲覧されても、受け手が意図した行動をしなければ良いコンテンツではない
- コンテンツには必ず目的がある（クリック、LINE登録、予約、無料体験参加など）
- その目的が達成できるように設計・制作する

### 人を動かすために越えるべき3つの壁

1. **読まない壁** -- 越え「続ける」必要がある（一度で終わりではない）
2. **納得しない壁** -- 越え「続ける」必要がある（一度で終わりではない）
3. **行動しない壁** -- 最終的にこの壁を越える

### エモーショナルリレー

- コンテンツはA地点（現在の状態）からB地点（狙った状態）まで見込み客の感情を連れていくもの
- B地点まで連れていけなければ、狙った行動は起きない
- 前後のコンテンツとの感情の乖離が大きいと離脱が発生する
- **シームレスな導線** = つなぎ目がなく、感情がリレーするように購入まで流れていく状態。これが実現できれば離脱率は非常に低くなる

## 4. VSL（ビデオセールスレター）

### VSLの定義と特性

- **VSL** = Video Sales Letter。30分〜1時間程度の販売動画
- **適用価格帯**: 5万円以下の商品・サービス
- 5万円超は、セミナーセールス/1対1セールス/ライブセールスなどコミュニケーション濃度の高い手法が王道
- **特徴**: 一方通行のコミュニケーション。見込み客は前向きな感情で見ていないため、小さなストレスで離脱する

### 反応率を高める6つのポイント

1. **冒頭で見続ける理由付けがあるか** -- 見込み客の問題点を鋭く指摘し、この動画を見ることで解決できると伝える。フックと「なぜこの動画を見るのか」を明確にする
2. **持続的に興味を引けているか** -- 序盤は1分に1回、中盤以降は3分に1回のペースでフックを入れる。「疑問 → 解消 → 疑問 → 解消」の流れを作る
3. **新規性のある解決策があるか** -- 既知の情報は離脱を招く。ターゲットにとって「新しい」「ひと味違う」と感じる打ち出しにする
4. **信頼性が構築されているか** -- 自分の実績、購入者の声、返金保証などの安心要素を提示する
5. **明確なオファーと行動喚起があるか** -- 終盤に強いオファーを設計する
6. **非言語情報で違和感がないか** -- 話すスピード、声のトーン、間の使い方、ライト調整、スライドの動きなどが成約率に影響する

### VSLの基本構成（3段階）

#### 第1段階: 興味付けパート

**目安時間**: 30分VSLなら5分以内、60分VSLなら15分以内
**重要度**: ここで約50%が決まる

**構成**:
1. **フック** -- 自己紹介の前に必ず入れる。冒頭で自分語りをしてはならない
2. **自己紹介** -- 1分、長くても2分以内
3. **実績紹介** -- 自分の実績 + クライアント実績

**フックの種類と例**:
- クイズ形式: 「このLPとこのLPはどちらが反応率が高かったでしょう？」
- 質問形式: 「完全未経験から広告で仕組み化するための全体像を解説していきますが、その前に1つ質問です」
- ストーリー形式: 「この男性はたった1年で広告からの収益が累計数億円になり、今もなお継続できています」

**フックの注意点**:
- 答えが当たり前だと見込み客はがっかりする。必ず意外性のあるものにする
- ネタが思いつかない場合は、顧客実績を出してその裏側を語る手法も有効

**自己紹介で伝える項目**:
- 名前と肩書き
- 専門分野とその経験
- なぜ見込み客にこの動画を届けるのか
- 理念や信念

**実績紹介のコツ**:
- 具体的な数字を混ぜる（「売上300%増加」「1000人以上を支援」「300人以上の指導実績」）
- 顧客事例はストーリー形式で語ると信頼度が上がる
- 他人の実績を入れることで「自分にもできそう」と思わせる

#### 第2段階: 本編パート

**重要度**: ここまでで約80〜90%が決まる

**意識すべき3つのこと**:
1. 全部を伝えずに全体の一部だと伝えること
2. 新規性のある解決策になっていること
3. NOを事前に潰しておくこと

**本編の8STEP構成**:

| STEP | 内容 | 目的 |
|------|------|------|
| 1 | 問題の深掘り | 興味付けで触れた問題を更に深掘りし、見込み客が思っている以上に深刻で複雑な問題と認識させる |
| 2 | 従来の解決策の問題点 | 今までの方法では不十分だと認識させる。失敗への共感を得つつ、新しい解決策への導入とする |
| 3 | 新しい解決策の提示 | 新規性のある解決策を示し「これなら解決できるかもしれない」と感じさせる |
| 4 | 解決策の説明 | 全体を見せた上で一部を超具体的に解説する。「自分にもできそうだが全部知らないとできない」状態を作る |
| 5 | 予想されるNOへの対処 | 最後ではなく途中でやらない理由を潰す（時間がない、自分にはできなさそう、リスクがあるのでは等） |
| 6 | 安心要素の提示 | 他人の成功事例、権威ある人からの評価を示して信頼性を構築する |
| 7 | 競合との違いを説明 | 序盤で独自性が十分出せていない場合に入れる（十分な場合は省略可） |
| 8 | 本編の総まとめと行動の重要性 | オファーへの架け橋。行動の重要性を説くことで、オファー時に行動しやすい状態を作る |

#### 第3段階: オファーパート

**定義**: オファー = 取引の条件。こちらが出す条件に対して「欲しい」と思ってもらうこと。
**前提**: オファーに至るまでに約90%は決まっている。成約に至らない場合はオファー前の段階で失敗していることがほとんど。

**オファーの6STEP構成**:

| STEP | 内容 | 詳細 |
|------|------|------|
| 1 | 正式な商品・サービスの紹介 | 商品コンセプトで全体像を理解させ、達成できるゴールを明確にした上で、具体的な内容を詳細に伝える |
| 2 | 総価値と実際の価格の提示 | 各カテゴリーごとに価値を定め、合算した総価値を提示した上で、パッケージ化した実際の価格を示す。価値と価格のバランスの正当性を与える |
| 3 | 安心要素の提示 | 返金保証、成果保証、サポート保証など。保証を入れるとCVR・売上が上がる。ただし強すぎると質の低い顧客が集まるリスクあり |
| 4 | 緊急性・限定性・希少性の提示 | 48時間限定、数量限定、限定特典、先着順など。行動しない壁を突破する心理トリガー。やりすぎると信頼度低下のリスクあり |
| 5 | CTA（Call To Action） | 「画面下のボタンから購入してください」「公式LINEで○○と送ってください」など、具体的な行動指示を明確に出す |
| 6 | Q&A | 想定される不安・質問に先回りして回答する。序盤でNOを十分潰していれば省略可 |

## 5. セミナーセールス

### VSLとの違い

| 項目 | VSL | セミナーセールス |
|------|-----|----------------|
| コミュニケーション | 一方通行 | 双方向（コメント・質問対応が可能） |
| パーソナライズ | 難しい | 質問に応じてその場で調整可能 |
| 適用価格帯 | 5万円以下 | 15万円超の高単価商品に推奨 |
| 自動化 | しやすい | リアルタイムが基本（オートウェビナーで自動化も可能） |
| デメリット | 聞く耳を持たれにくい | 参加人数の制限、時間的制約がある |

### セミナーセールスでやるべき2つのこと

1. **エネルギーを上げる** -- 新しい解決策で得られる未来を「欲しい」と思ってもらう。欲しい度合い（エネルギー）をどんどん高める
2. **ハードルを下げる** -- 解決策を手に入れることへの不安（金銭的・時間的・能力的コスト感）を取り除く。他人の実績提示、価格に対する価値の正当性などで対処する

### セミナーのマインドセット

- 売ることだけを考えずに顧客の問題解決を行う
- 顧客に媚びず対等な目線で話す
- 自分が話したいことばかり話さない
- **メラビアンの法則**: 言語7%、聴覚38%、視覚55%。「何を言うか」だけでなく「誰が言うか」が重要

### セミナーセールスの基本構成（3段階）

VSLと同じく「興味付け → 本編 → オファー」の3段階。

#### 第1段階: 興味付けパート（20〜30分）

**ゴール**: 本編パートに入ってきてもらうこと

**見込み客が達成すべき4つの状態**:
1. 発信者に対して信頼している
2. 自分の理想の状態が明確である
3. 自分の問題点について理解している
4. 本編の内容に期待している

**興味付けパートの構成**:

**(1) ラポール形成と問題提起**
- セミナーセールスの約50%はここで決まる
- ラポール形成 = 信頼構築。相手からアクションしてもらうことが重要
- 簡単な問いかけに対してコメントしてもらい、そのコメントをしっかり拾う
- 問いかけ例:
  - 「今日はどこから視聴されていますか？」
  - 「今日のセミナーで聞きたいことは何ですか？」
  - 「どんな目標を掲げていますか？」
  - 「その目標に対してこれまでどんなことをしてきたのか？」
- 浅い質問からセミナー内容に関係のある質問に移行させ、問題提起に繋げる
- 問題提起: これまでの努力では悩みを解決できないかもしれないと気付いてもらう。事前リサーチで来そうなコメントを予想しておく

**(2) セミナー価値の提示**
- 悩みを解決する新しい手段が見つかることを伝える
- その解決策がなぜ良いのかを証明する客観的事実を伝える
- セミナーを聞くことによるベネフィットを提示する

**(3) 成功事例と問題意識の強化**
- 成功者と現在のセミナー参加者の違いを明確にする
- ポジショニングマップなど図を用いて可視化する
- 成功者が実際に行っていること、失敗者との違い、失敗の延長線上には答えがないことを再確認する

**(4) 講師の紹介**
- 最初から自己紹介はしない（広告集客の場合、参加者は発信者ではなく内容に興味がある）
- 興味付けを行った後に自己紹介を入れる
- 権威性をしっかり出すことがカギ

**(5) 本編への橋渡し**
- 見込み客に目指す理想をコメント等で宣言してもらう（後々効いてくる）
- 最低でもYesと答えられる質問を問いかける

#### 第2段階: 本編パート

VSLの本編と基本的に同じ考え方。以下の5要素で構成する:

1. **解決策の説明**
2. **予想されるNOへの対処**
3. **安心要素の提示**
4. **競合との違いを説明**
5. **本編の総まとめと行動の重要性**

#### 第3段階: オファーパート

VSLのオファーと基本的に同じ。セミナーセールスの場合、コメントで質問回答ができるため不安を取り除きやすい利点がある。

## 6. 販売導線の仕組み化

### 仕組み化と自動化の違い

| 概念 | 定義 |
|------|------|
| **仕組み化** | 集客した見込み客に対して商品・サービスを届ける流れを形にすること |
| **自動化** | システムを活用して、仕組み化した流れを人のリソースなしで回すこと |

- 自動化には適切な仕組み化が必要不可欠
- 仕組み化 ≠ 自動化。人のリソースが必要かどうかが違い

### 自動化の適切なタイミング

- 集客した見込み客に効率的に商品を届ける見込みがあり、自動化しても数字が大きく落ち込まない時がベストタイミング
- 楽をしたいだけで自動化しても上手くいかない。良くない仕組みのまま自動化しても購入に至らない
- **適切な循環**: 泥臭い仕組みを行う → 効率が高まる → 自動化 → リソースが空く → 泥臭い別の施策を行う → 事業が拡大

### 販売導線ツール比較

| ツール | 特徴 |
|--------|------|
| **UTAGE** | オールインワン（メール配信・LINE配信・ページ制作・ファネル制作）。月額約2万円。コスパが良く一括管理が可能 |
| **Lステップ** | 分析機能はUTAGEより優れる。ただし公式LINE1つにつき料金がかかり、メール・LP制作に別ツールが必要でランニングコストが高い |

UTAGEが触れるようになればLステップもすぐに使える（設定が類似）。

### UTAGEの主要機能

| 機能 | 内容 |
|------|------|
| メール配信 | 通数無制限 |
| ステップ配信 | LINEとメルマガの両方送信可能。時間軸のズレを防止 |
| ファネル構築 | ノーコードでLP作成可能。LP作り放題。ABテストもしやすい |
| 商品管理 | 決済はテレコム、アクアゲート、ユニバペイ、ストライプの4社対応。推奨はユニバペイ（審査は厳しい） |
| 会員サイト | 動画保管庫として活用可能。読者の進捗管理も可能。動画・PDF・画像・テキストに対応 |
| ウェビナー機能 | セミナーセールスをリアルタイム感のある演出で自動化配信できる |
| イベント機能 | 個別相談やセミナーの予約管理。人数設定なども細かく設定可能 |

## セールス手法の選択基準（早見表）

| 商品価格帯 | 推奨セールス手法 |
|------------|----------------|
| 5万円以下 | VSL |
| 5万円〜15万円 | VSL or セミナーセールス |
| 15万円超 | セミナーセールス / 1対1セールス / ライブセールス |

**判断基準**: コミュニケーションの密度が高いほど高単価商品の販売に適する。

## チェックリスト

### VSL制作チェックリスト

- [ ] 冒頭にフック（クイズ/質問/ストーリー）が入っているか
- [ ] 自己紹介は2分以内にまとめているか
- [ ] 具体的な数字を含む実績紹介があるか
- [ ] 顧客事例がストーリー形式で入っているか
- [ ] 全体像を見せた上で一部を具体的に解説しているか
- [ ] 想定されるNOを本編中に潰しているか
- [ ] 安心要素（成功事例・権威評価）が入っているか
- [ ] 商品の総価値と実際の価格が提示されているか
- [ ] 返金保証・成果保証等の保証があるか
- [ ] 緊急性・限定性を入れているか（やりすぎていないか）
- [ ] CTAは具体的で分かりやすいか
- [ ] 非言語情報（スピード・トーン・間・映像）に違和感がないか

### セミナーセールスチェックリスト

- [ ] ラポール形成のための問いかけを準備しているか
