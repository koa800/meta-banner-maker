# 広告クリエイティブ制作

広告クリエイティブ（テキスト・バナー・動画）の設計思想、制作フレームワーク、品質判断基準をまとめたナレッジ。クリエイティブの企画・制作・改善のすべてのフェーズで参照する。

---

## 1. マインドセット

広告クリエイティブ制作に入る前に、以下の4つを常に意識する。

### 1-1. CPAではなくROASを見る

- クリエイティブは顧客の入り口であり、集まる顧客層を決定する
- CPAを下げることは容易だが、商品・サービスとの関連性が低い顧客を集めてしまうリスクがある
- 判断基準は常にROAS（広告費用対効果）
- 例: CPA 1,000円で獲得 → 成約コスト10万円 vs CPA 2,000円で獲得 → 8万円の商品を購入。後者の方がROASが高い

### 1-2. ターゲットの理解を怠らない

- クリエイティブは最終的に「人」が見るもの
- ターゲットの生活環境、行動習慣、利用SNS、悩み、願望を具体的に理解する
- 顧客の視点からクリエイティブを設計する

### 1-3. 仮説検証から逃げない

- 最初から成功するクリエイティブは作れない
- 仮説を立て、検証を繰り返し、精度を上げていくプロセスが不可欠
- 失敗を恐れず試行錯誤を続ける

### 1-4. 正しくお金を使う

- 無駄な広告費を避けるために知識を深める
- 一方で、広告費を抑えすぎることも機会損失になる
- トレンドに乗って集客を加速させるべきタイミングでは、思い切って投入する
- 広告は集客をコントロールできる手段。タイミングを見極めて効果的に資金を投入する

## 2. 広告クリエイティブの定義と基本構造

### 広告クリエイティブとは

各広告媒体で配信するための制作物の総称。クリエイティブのクオリティによってROASが大きく変動する。同じ導線・同じLPでも、クリエイティブが変わるだけでROASが100〜200%変わることがある。

### 広告クリエイティブのゴール

**目的に対して「3つの壁」を越えること。**

見込み客が商品を購入するまでのステップ:

```
認知 → 興味 → 欲求 → 比較検討 → 購入
```

クリエイティブはこの流れの「入り口」を担う。クリエイティブの種類によって、認知〜興味までを担う場合もあれば、欲求の段階まで担う場合もある。

### 良い広告クリエイティブの定義

- 同じターゲット層における離脱率が低いクリエイティブ
- 同じターゲット層でCPAが安くなるクリエイティブ
- 結果としてROASが高くなるクリエイティブ

### クリエイティブの本質

**インプレッションを買って人を動かすこと。** 広告クリエイティブは人を「ある地点」から「ある地点」へ動かすためのコンテンツ。どういった見込み客をどのように動かしたいかを考えることで、良いクリエイティブが完成する。

## 3. 広告クリエイティブの種類

### テキスト広告

- リスティング広告やX（旧Twitter）広告など、文字だけで出稿する形式
- テキスト広告 → LP → CV の流れが基本

### バナー広告

- 画像1枚で構成される広告形式（Meta広告など）
- **メリット**: 制作コストが低く、PDCAを回しやすい。コンセプトやコピーのテストに最適
- **役割**: 主に認知促進・興味喚起。教育力は高くないため、LPでの教育が必要
- **活用法**: コンセプトテスト → 効果があったものを動画化して強化

### 動画広告

- Meta広告、TikTok広告、YouTube広告など動画配信可能な全媒体で利用
- **メリット**: 情報量が圧倒的に多く、教育力が高い。動画内でオファーへの欲求を喚起できる
- **デメリット**: 撮影・編集にコストがかかる
- **役割**: 認知〜欲求までを1つで完結させることが可能。LPの教育力が小さくてもCVが発生しやすい

### バナー広告と動画広告の比較

| 観点 | バナー広告 | 動画広告 |
|------|----------|----------|
| 教育力 | 低い（認知〜興味まで） | 高い（認知〜欲求まで） |
| 制作コスト | 低い | 高い |
| PDCAの速度 | 速い | 遅い |
| LPへの依存度 | 高い（LP側で教育が必要） | 低い（動画内で教育可能） |
| 推奨用途 | コンセプト・コピーのテスト | CVR最大化 |

### PDCAサイクルの活用

バナー広告と動画広告を組み合わせて活用する:

1. バナー広告でコンセプト・コピーをテスト
2. 反応の良いコピー・訴求を特定
3. そのコピーを動画広告にしてCVR（コンバージョン率）を最大化

## 4. 3つの壁フレームワーク（詳細）

広告クリエイティブの品質を決める最重要フレームワーク。この3つの壁を突破できるかどうかが、クリエイティブの成否を分ける。

### 第1の壁: 読まない・見ない壁

**定義**: 見込み客がクリエイティブに気づかない、または気づいても見るのをやめてしまう壁。

**重要な認識**: この壁は一度超えて終わりではなく、**超え続けるもの**。最初に興味を引くだけでなく、興味を惹き続けることが重要。

#### 突破手段（4つ）

| 手段 | 内容 | 例 |
|------|------|------|
| 文字 | ターゲットが無意識に反応する脳内ワードを使う | 業界用語、悩みに直結するキーワード |
| ビジュアル | 商品・人・世界観で興味を引く | 目を引く画像、ターゲットが共感する人物 |
| 企画 | ドッキリ・クイズ等のフォーマットで興味を維持 | 「9割が知らない〇〇」「〇〇 vs △△」 |
| 音 | 動画広告で音によって興味を引き続ける | BGM、効果音、声のトーン |

#### 訴求ワードの2分類

- **脳内ワード**: 見込み客の頭の中に既に存在しているワード（例: SNS、広告、ホットペッパービューティー）
- **興味ワード**: 多くの人が反応しやすいフックになるワード（例: 「新常識」「9割が知らない」「実は〇〇」）

### 第2の壁: 納得しない壁

**定義**: クリエイティブを見ていても、行動するための妥当性が感じられず離脱する壁。

**突破の原則**: 目的に対して行動するための説得力を、心理トリガーを用いて高める。

#### 突破手段（4つ）

**A. 信頼度を高める**

信頼には2種類ある:

1. **発信者への信頼**
   - 権威（実績・肩書き・専門性）を伝える
   - 親近感（仲の良い人、共感できる人）を演出する
   - 原則: 「何を言うか」の影響度は「誰が言うか」によって決まる
   - 人が納得するのは「偉い人」か「仲の良い人」から言われたとき

2. **オファーへの信頼**
   - 社会的証明を活用（成功事例の共有、多数の事例、レビュー）
   - 「みんなが良いと思っているもの」「みんなが成果を出しているもの」は良いものだと判断される

**B. 目的を明確にする（オファーを受け取る意味の付与）**

2つのアプローチ:

1. **ベネフィットを提示する**（プラスの目的）
   - ベネフィット = メリットがもたらすプラスの影響
   - ベネフィットは1つではなく多数存在する。その中からターゲットにとって最適なものを選ぶ
   - 例（カメラ）: 「写真が撮れる」ではなく「子どもとの大切な時間を写真に残せる」「家族との思い出を増やせる」
   - 最適なベネフィットにより「○○のために購入する」という目的が明確になる

2. **受け取らないことによる損失を伝える**（マイナスを避ける目的）
   - 人は得られる報酬よりも損失を回避したい生き物（損失回避バイアス）
   - オファーを受け取らないことでどれだけの損失があるかをイメージさせる
   - **注意**: 恐怖訴求のやり過ぎはネガティブマインドな見込み客を集めやすい。購入後のトラブルにもつながる
   - 判断基準: 「どういったお客様を集めたいか」から逆算した訴求にする

**C. 問題を意識させる**

2つのアプローチ:

1. **顕在的な問題を再認識させる**
   - 普段抱えている問題を「思い出させる瞬間」を作る
   - 方法: 感情の言語化、悩みの言語化
   - 効果: 分かりやすい言葉で共感を得る → 信頼関係の構築 → その後の話を聞いてもらいやすくなる
   - 解決策提示のティーアップにもなる

2. **潜在的な問題を顕在化させる**
   - まだ認識していない問題を掘り起こして認識させる
   - 見込み客にとって「不都合だが無視できない情報」を提示する
   - 例: 「フォロワー数と収益は関係ない」→ フォロワーを伸ばすことが常識の人にとって、自分のやり方に問題があると気づくきっかけ
   - 問題が認識できれば、解決策（オファー）提示のティーアップになる

**D. オファーを提示・強化する**

必要なこと:

1. **解決策（オファー）を提示する**
   - オファーの名前がシンプルで分かりやすいことが重要。名前が魅力的であれば数字が上がる

2. **解決策の魅力度を高める**
   - **効果に対する納得感を与える**: なぜ効くのか、具体的な根拠を示す（例: 「○○成分が配合」）
   - **従来の解決策との違いを提示する**: 新規性のあるアプローチであることを伝え、「自分の悩みを解決できるかもしれない」という感情を引き出す
   - **対立構造を作り比較する**: 何かと比べることで良い・悪いの判断を促す（例: 「動画編集は月5万まで再現性あり。プログラミングなら収益を大きく伸ばせる可能性が高い」）

### 第3の壁: 行動しない壁

**定義**: オファーの内容が良いと思っても「今受け取らなくてもいい」と後回しにする壁。

#### 突破手段

1. **限定性・希少性・緊急性を付与する**
   - 「30名限定」「先着○○名まで」「期間限定」
   - 「本気の人しか受け取らないでください」（商品への自信を打ち出す/あえて突き放す）
   - やり過ぎると逆効果。さらっと伝える程度が推奨

2. **CTAを明確に伝える**
   - 受け取りたいと思っていても、方法が分からなければ行動しない
   - 具体的な行動指示を必ず入れる
   - 例: 「下のボタンをタップして受け取ってください」「右にスワイプして次に進んでください」

## 5. LPとの一貫性（絶対ルール）

**広告クリエイティブ制作で最も守るべきルール: LPとの一貫性を保つこと。**

一貫性がなければメッセージがブレ、見込み客はストレスを感じて離脱する。

### 一貫性を保つ2つのアプローチ

#### 統合型

- 広告クリエイティブで**具体的な内容**を訴求し、LPでは**1つ抽象度が上がったテーマ**にする
- 例: LP「集客方法」→ CR「SNS集客の秘訣」「広告で新規集客2倍」「メール施策の効果」

| | 広告クリエイティブ | LP |
|--|--|--|
| 抽象度 | 低い（具体） | 高い（上位概念） |
| 例 | 「毎日10分のヨガで体が変わる」 | 「健康的な生活習慣の築き方」 |
| 例 | 「バランスの良い食事で免疫力UP」 | 「健康的な生活習慣の築き方」 |

- **メリット**: 複数の具体的な切り口でCRを展開でき、見込み客の母数を広げやすい
- **適用場面**: 規模が大きくなり、見込み客の母数を広げる必要がある段階

#### 平行型

- 広告クリエイティブとLPの**テーマの抽象度が同じ**
- 例: LP「30日で英語力を向上させる学習法」→ CR「1ヶ月で英語力がアップする効果的な学習法」「4週間で英語スキルを磨く秘訣」

| | 広告クリエイティブ | LP |
|--|--|--|
| 抽象度 | 同じ | 同じ |
| 例 | 「CPAが1000円下がった広告集客法」 | 「新規集客が2倍になった広告集客方法」 |

- **メリット**: シンプルで作りやすい
- **注意**: 抽象度が高すぎると見込み客を捉えられない（「起業」「ビジネス」は広すぎる）。狭い領域で尖らせる
- **適用場面**: 初期段階、テーマが明確な場合

### 一貫性チェックリスト

- [ ] CRとLPのテーマは同じ上位概念に属しているか
- [ ] CRで伝えた訴求内容はLPの中で回収されているか
- [ ] CRとLPの抽象度の差が大きすぎないか
- [ ] CRで触れていない別のテーマがLPに混在していないか

### NGパターン

- **メッセージのずれ**: LP「プログラミングでキャリアアップ」なのにCR「簡単な副業で月10万円」
- **抽象度の乖離**: LP「効果的なマーケティング戦略」なのにCR「Instagram投稿の最適な時間帯」「SEO対策のメタタグの書き方」

## 6. 制作フロー

### バナー広告クリエイティブ制作フロー（6ステップ）

1. **ゴール設定**: どのターゲット層にどう行動してほしいかを明確にする
2. **訴求ワードリサーチ**: ターゲットが反応する脳内ワード・興味ワードを洗い出す
3. **コンセプトメイク**: 誰に何を伝えるかを明確にし、訴求ワードを活用して強力なフレーズを作る
4. **フォーマットリサーチ**: コンセプトをどのように伝えるかを決める
5. **コピー作成**: 訴求フレーズを元にコピーを作成
6. **デザイン実装**: 画像として仕上げる

### 動画広告クリエイティブ制作フロー

基本フローはバナー広告と同じだが、以下の工程が追加される:

1. ゴール設定
2. 訴求ワードリサーチ
3. コンセプトメイク
4. フォーマットリサーチ（動画特有: 演者一人で話す形式、対話形式、スライド形式など）
5. **台本作成**（追加工程）
6. **撮影**（追加工程 / フォーマットによっては不要）
7. **編集**（追加工程）

## 7. クリエイティブ制作チェックリスト

### 企画段階

- [ ] ゴール（目的）は明確か
- [ ] ターゲットは具体的に定義できているか
- [ ] ROASの観点で評価する準備ができているか

### 3つの壁の突破設計

- [ ] 第1の壁（読まない・見ない）: 興味を引き続ける仕掛けがあるか（文字/ビジュアル/企画/音）
- [ ] 第2の壁（納得しない）: 信頼・目的・問題意識・オファーの4要素が入っているか
- [ ] 第3の壁（行動しない）: 限定性とCTAが明確か

### 品質チェック

- [ ] LPとの一貫性は保たれているか（統合型 or 平行型）
- [ ] ターゲットの脳内ワード・興味ワードが使われているか
- [ ] ベネフィットが「メリット」ではなく「メリットがもたらすプラスの影響」で表現されているか
- [ ] 恐怖訴求が過度になっていないか
- [ ] オファーの名前はシンプルで分かりやすいか
