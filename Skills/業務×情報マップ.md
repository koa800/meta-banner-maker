### Lステップ詳細
- **目的**: スキルプラス販売のマーケティング全般（集客→LINE登録→購入の導線）+ 受講生サポート
- **実務**: 配信シナリオ設定 / テンプレート作成 / 流入経路作成 / タグ設定 → 全工程を甲原さんが自分で実行
- **データ管理**: タグ情報を中心にデータ管理
- **アカウント数**: マーケティング活用で約15個運用
- **CS用アカウント**: 3つ（スキルプラスサポートLINE / エリート / プライム）→ 今後「スキルプラスサポートLINE」に統合予定
- **ログインURL**: `https://manager.linestep.net/account/login`（1Password自動入力 + reCAPTCHA突破）

## 受講生CS対応

- **本質**: テクニカルサポートではない。**広告・マーケティングのアクションに対するコンサル・添削が中心**
- **よくある質問**: 「添削してください」「こういう案件があるのですが、どうすればいいですか？」等
- **テクニカル問題**: SuccessLearning.ai の中で解決済み。基本的に来ない
- **情報源**:
  1. 甲原さんの頭の中のドメイン知識（広告・マーケ実務）← これが最大のボトルネック
  2. 質問回答のスプレッドシート（過去の回答蓄積。シートID未確認）
  3. Lステップやドキュメントで共有した過去の回答まとめ
  4. **過去のCS回答はLoom動画で録画されている** → 動画学習パイプラインで取り込み可能
- **対応フロー**: 受講生がLステップ or SuccessLearningで質問投稿 → 甲原がスプレッドシートで確認 → Lステップで返答
- **重要**: 商品マスタはCS対応に無関係

## 広告クリエイティブ制作

### プロセス
1. **企画段階**: 過去クリエイティブデータ・SNS・競合広告リサーチ / 「動画広告分析プロ」等のリサーチツール / Instagram・X・YouTubeの伸び投稿分析 → 自分の企画に落とし込み
2. **台本作成**: 広告オファー内容と情報の整理 / ターゲット理解 / 過去の成功パターンのトレース・オマージュ
3. **ツール活用**: 画像=Gemini Nano(Google Pixel等オンデバイスAI)+最新ツール / 動画=Kling AI・Vidu等を目的別に使い分け
4. **編集・納品**: 動画編集者（はたけさん等）に依頼 / 全体フローは甲原さんが把握・管理

## 既存データソースの役割

| データソース | 業務 | 役割 | CSで使うか |
|------------|------|------|-----------|
| 商品マスタ | 商品管理 | 全商品の台帳。新商品は必ず記載（運用ルール） | **使わない** |
| 口コミデータ（SNS監視シート） | 改善・施策評価 | ネガティブ要因の潰し込み + ポジティブ施策の効果測定 | 使わない |
| イベント運用データ | イベント管理 | 参加者数・満足度の集計 | 使わない |
| メディア掲載・露出データ | 広報・広告 | 露出記録 + 二次利用（広告運用での再活用）+ 社内問い合わせ対応 | 使わない |
| インフルエンサーキャスティング | 広報施策 | アプローチ履歴 + 口コミ発生元の追跡 | 使わない |
| KPIデータ（Looker） | 経営判断 | 集客数・予約数・売上・入会・解約の最新値。常に即取り出し可能に | 使わない |
| Addnessゴールタスク | 全社マネジメント | 全メンバーのアクション可視化 + 進捗・解像度の確認 | 使わない |
| 質問回答スプレッドシート | 受講生CS | 過去の回答蓄積 | **使う** |
| Lステップ/ドキュメントの回答まとめ | 受講生CS | 過去回答の共有用 | **使う** |
| Loom動画（CS回答） | 受講生CS | 甲原さんの過去の回答録画 | **使う** |
