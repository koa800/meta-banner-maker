# 販売導線設計の基本

## このスキルについて

見込み客に商品・サービスを届けるための販売導線（セールスファネル）を設計・構築するための体系的なナレッジ。定義、設計フレームワーク、販売方法の選び方、導線の種類、強化施策、心理トリガーまでを網羅する。

---

## 1. 販売導線とは

### 定義

- **販売導線** = 複数の販売経路が統合されたもの
- **販売経路** = 商品・サービスをユーザーに届ける1つのルート
- 販売導線は「1つの商品の届き方」を設計するもの

### 販売導線 vs ビジネスモデル

| 概念 | 範囲 | 具体例（歯医者） |
|------|------|----------------|
| 販売導線 | 1つの商品の販売プロセス | 矯正の販売プロセス、ホワイトニングの販売プロセス |
| ビジネスモデル | 事業全体の仕組み | 矯正・ホワイトニング・インプラント等を統合した歯科サービス全体 |

- 事業には複数の販売導線があり、それらを統合して事業全体の仕組みを作っているのがビジネスモデル

### 販売導線でやるべき2つのこと

1. **見込み客レベルを冷たい状態から熱い状態に引き上げる**
2. **熱い見込み客に対して商品・サービスをオファーする**

---

## 2. 見込み客の3段階レベル

| レベル | 状態 | 具体例（ダイエット） |
|--------|------|-------------------|
| 冷たい | 欲求を認知していない、または欲求はあるが問題を認知していない | 痩せたいと思っていない、または太っている原因がわからない |
| 温かい | 欲求・問題は認知しているが解決策を認知していない | 食べ過ぎが原因とわかっているが、生活リズムを崩さず痩せる方法がわからない |
| 熱い | 欲求・問題・解決策を全て認知している | 痩せたい欲求も、原因も、解決方法もすでに知っている |

- 商品が売れる状態 = 熱い見込み客が多い状態
- 販売導線内の施策は以下の3つに集約される
  1. 冷たい → 温かい に引き上げる施策
  2. 温かい → 熱い に引き上げる施策
  3. 熱い見込み客へのオファー

## 3. SNSと広告の販売導線の違い

| 項目 | SNS | 広告 |
|------|-----|------|
| リストイン時の見込み客レベル | 熱い（SNS上で教育済み） | 冷たい（初見ユーザーが多い） |
| 教育の場所 | SNS上（投稿・ストーリーズ等） | LINE・メール等の導線内 |
| 導線の複雑さ | 簡易的でも売れる | 冷→熱への教育プロセスが必要 |

### 広告導線のポイント

- **初見ユーザーにも売れる導線**を構築することが最重要
- 理想はエスカレーター型：乗ればそのまま自動で商品販売まで到達する導線
- 階段型（1ステップずつ負荷がかかる）ではなく、シームレスに設計する

### 広告導線を先に構築するメリット

- 初見ユーザーに売れる導線があれば、SNS・HP・SEO等あらゆる集客媒体に展開可能
- SNS導線をそのまま広告に転用しても成果が出ないケースが多い
- 広告導線 → 他媒体へのスケールは容易だが、逆は困難

## 4. 販売導線で意識すること

### 核心

- 集客した見込み客が商品を購入するまでの**離脱率をできる限りなくす**こと

### 見込み客が離脱する主な理由

- 発信者への信頼がない
- 商品で得たい未来がイメージできない
- 自分の悩みがその商品で解決できると思っていない
- 解決できると認識しているが、お金を使う決断ができない
- 今やる必要性を感じていない

### LTVの重要性

- **売上 = 集客 x LTV**
- **LTV = 成約率 x 商品単価 x リピート率**
- 販売導線の強化により成約率・リピート率が上がり、LTVが向上する

### LTVが高くなる3つのメリット

1. 広告予算をより多くかけられる → 集客数増加
2. 競合の参入で広告費が上がっても耐えられる
3. 利益額が上がり、仕組み化の強化に投資できる

---

## 5. 販売導線を強化する3つの施策

### 5-1. コミュニケーションの形を強化する

- コミュニケーション密度が高いほど効果が上がる
- 強さの順: テキスト < 画像 < 動画 < 個別返信 < セミナー < 個別相談
- 密なコミュニケーションは信頼構築とコンテンツ視聴率の向上に寄与する

### 5-2. セールス回数を増やす

- シンプルにセールス回数を増やすだけでCVRが上がる
- 例: メール3通 → 5通に増加、期間限定キャンペーンでの再販売
- 多すぎると信頼低下のリスクがあるためバランスが必要

### 5-3. 商品数を増やす

| 種類 | 定義 | 具体例 |
|------|------|--------|
| アップセル | メイン商品のアップグレード版 | ホテルの部屋のグレードアップ提案 |
| ダウンセル | メイン商品のダウングレード版 | 購入しなかった人への廉価版提案 |
| クロスセル | メイン商品の関連商品 | 飲食店でのドリンク・サラダの追加提案 |

## 6. 心理トリガー

### 前提

- 人は**感情で物が欲しくなり、論理で購入を正当化**する
- 商品を買ってもらうために必要な2つのこと
  1. **信頼してもらう**こと
  2. **NOがない状態を作る**こと（買わない理由を全て潰す）

### 6つの心理トリガー

| トリガー | 原理 | 活用例 |
|----------|------|--------|
| 権威性 | 偉い人・地位が高い人の言葉は説得力が高い | 有名人の推薦、専門家の監修、実績の提示 |
| 返報性 | 何かをしてもらうとお返ししたくなる | 無料特典の提供、手書きメッセージ |
| 社会的証明 | みんながやっている＝良いものと判断する | 口コミ、高評価、行列、お客様の実績 |
| 信頼 | 偉い人か親しい人のことは信頼しやすい | 関係構築、親近感のある発信 |
| 限定性 | 得るものより失う恐怖の方が強い | 期間限定、数量限定、今だけ特典 |
| 希少性 | 手に入りづらいものに価値を感じる | 予約困難、少数精鋭、限定募集 |

## 7. 販売導線設計の基本知識

### 設計の定義

- 販売導線における設計 = **見込み客とのコミュニケーション方法を決定すること**
- テキスト・画像・動画・セミナー等は全て発信者と見込み客のコミュニケーションツール

### 設計のゴール

- **シームレスな顧客体験をコミュニケーションコンテンツ単位で描く**こと
- シームレス = つなぎ目がない状態
- コンセプトに沿った一貫性のあるメッセージを伝え、滑らかに商品販売まで導く

### 設計がしっかりしていると

- 一貫性のある顧客体験 → 満足度向上
- 離脱率の低下
- ブランド価値の向上
- 無駄なリソース削減 → コスト削減

### 設計が不十分だと

- 離脱率が高くなる
- 信頼度が低下する
- 時間・お金が足りなくなる

### 販売導線の構成要素

| 種類 | 内容 | 特徴 |
|------|------|------|
| 非リアルタイムコミュニケーション | 音声、画像、動画、テキスト | 自動化可能 |
| リアルタイムコミュニケーション | オフライン/オンラインのセミナー、個別相談、展示会、イベント | 教育力が高い |

## 8. 販売導線設計の6ステップ

### 全体フロー

1. **商品の販売方法を1つ決める**
2. **その販売方法に繋げるための施策を洗い出す**
3. **洗い出した施策を2軸で評価する**
4. **低コストかつ効果が高い施策を最小構成で構成する**
5. **残りの施策を肉付けしていく**
6. **販売方法を追加して再度1-5の流れを組む**

### STEP 1: 商品の販売方法を1つ決める

販売方法を決める2つの要素:

| 要素 | 影響 |
|------|------|
| 商品単価（最重要） | 高単価 → 個別相談・セミナー / 低単価 → テキスト販売も可 |
| 見込み客レベル | レベルが高ければ、密なコミュニケーションなしでも販売可能 |

**販売方法の種類:**

- **オフライン販売**: 1対1販売、セミナー、展示会、イベント
- **オンライン販売**: 個別相談、セミナー、電話、メール、DM、HP
- **その他**: 紹介

**成約率が高い順（コミュニケーション密度順）:**
個別相談（1対1）> セミナー（1対多）> 電話 > テキスト（メール・DM・HP）

**販売方法の決め方:**
- コスト対効果が最大化するものを選ぶ
- 組織形態・営業リソース・スキルを考慮する
- 競合が使っている販売方法を参考にするのが失敗しにくい

**商品単価別の目安:**

| 商品単価 | 推奨販売方法 |
|----------|------------|
| 高単価（数十万円以上） | 個別相談 |
| 中単価（20-30万円程度） | セミナーでも可 |
| 低単価（数千円） | テキスト販売可 |

### STEP 2: 販売方法に繋げるための施策を洗い出す

- 販売導線の種類は無限にあるため、**競合の販売導線を分析する力**が重要
- 広告集客を行っている競合から学ぶのが有効（SNS導線は簡易的すぎて広告に転用不可）
- 最も効果的な方法: 自分自身で競合の導線を体験する（LINE登録 → 無料特典 → セミナー → オファーの流れ）

**広告集客で使われる高単価（15万円以上）向け販売導線:**

1. セールスレター型
2. 直個別相談型
3. 1話の個別相談型
4. 3話の個別相談型
5. ウェブセミナー型

### STEP 3: 洗い出した施策を2軸で評価する

- **X軸（横）**: コスト（右に行くほど低コスト） - 制作費用、時間、労力を総合的に考慮
- **Y軸（縦）**: 効果（上に行くほど高い効果） - 競合の成功事例を参考に判断
- 主観的な評価でもよいので各施策を4象限に分類する
- 広告で成功している競合は強力な販売導線を持っていることが多い

### STEP 4: 低コストかつ効果が高い施策を最小構成で構成する

- **最小構成** = 1つの販売経路として成立する最低限の状態
- 例: LINE登録 → 動画1本視聴 → 個別相談オファー
- **最重要ポイント: スピード**
  - 早く構築し、早くテストすることが最優先
  - 効果が高いが重い施策は後から追加する
  - 1ヶ月でできることを3ヶ月かけるのは機会損失
- 頭の中ではなく、目に見える形で図にまとめる

### STEP 5: 残りの施策を肉付けする

- 1つの販売方法での離脱率をできるだけ下げるための設計図を完成させる
- STEP 3で右上（低コスト・高効果）に分類した施策から順に追加する

**見込み客の離脱の3大原因と対策:**

| 離脱原因 | 対策 |
|----------|------|
| 読まれていない | リマインドの強化 |
| 納得していない | 訴求と演出方法を変えて多角的にアプローチ |
| 行動していない | CTAの改善、限定性・緊急性の付与 |

### STEP 6: 販売方法を追加して再度1-5を組む

- 年商5,000万円を超えた段階で新しい販売方法の追加が有効
- 既存の販売導線と統合する形で設計する（別々に考えない）
- 例: 個別相談で成約しなかった見込み客 → セミナーで再オファー
- メインの販売方法を補填するサブ的な販売方法として組み込む
- オートウェビナー等を活用し、自動化された導線を追加することも可能

## 9. 売れる販売導線構築の5STEP

### 全体フロー

| STEP | 名称 | ゴール |
|------|------|--------|
| 1 | 市場リサーチ | 顧客インサイトを掴み、競合の供給の届き方を明確にする |
| 2 | コンセプトメイク | 販売導線で伝えたいメッセージの軸を決める |
| 3 | 設計 | シームレスな顧客体験をコンテンツ単位で描く |
| 4 | コンテンツ制作 | 設計図を元に1つ1つのコンテンツを作り切る |
| 5 | 仕組み化 | 販売導線を通る見込み客に商品を届ける体制を整える |

### STEP 1: 市場リサーチ

- **ペルソナ分析**: インサイトを掴み「誰に」届けるかを決める
- **競合分析**: 競合がどのように商品を届けているかを明確にする

### STEP 2: コンセプトメイク

- 課題解決の方向性を決める
- 「誰に何を伝えるのか」をまとめる
- 例: 「枝葉の情報に踊らされるSNS発信者に対し、アルゴリズムに左右されない本質のSNSマーケティングを提供」

### STEP 3: 設計

- 商品の販売方法を決める
- 見込み客とのコミュニケーション方法を設計する
- エスカレーター型の滑らかな導線を描く

### STEP 4: コンテンツ制作

- テキスト、LP、セミナースライド等を実際に作成する
- **8割の完成度でも早く作ることを優先**する

### STEP 5: 仕組み化

- Lステップ、UTAGE等のツールを活用し、コンテンツを繋げて自動化する

## 10. オファーのタイミング設計

### よくある間違い

- 教育 → 教育 → 教育 → ... → 最後にオファー
- これでは序盤で熱い見込み客になった層が離脱してしまう

### 正しい設計

- 教育 → **オファー** → 教育 → 教育 → **オファー**
- 熱い見込み客には早めにオファーをかける

### 見込み客が導線内で取る2つの動き

1. 冷たい状態 → 熱い方向への移行
2. 販売経路からの離脱

### 離脱が起こるケース

- 既に熱い見込み客に過剰な教育をしてしまい、コンテンツが見られなくなる
- 冷たい状態から熱い方向へ動かすことに失敗した
