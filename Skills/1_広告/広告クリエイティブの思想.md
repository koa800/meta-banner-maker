# 広告クリエイティブの思想

広告集客とクリエイティブに関する設計思想・哲学・品質判断基準の原則集。

## 1. マインドセット

広告クリエイティブに向き合う前に、以下を常に意識する。

### 1-1. CPAではなくROASを見る

- クリエイティブは顧客の入り口であり、集まる顧客層を決定する
- CPAを下げることは容易だが、商品・サービスとの関連性が低い顧客を集めてしまうリスクがある
- 判断基準は常にROAS（広告費用対効果）
- 例: CPA 1,000円で獲得 → 成約コスト10万円 vs CPA 2,000円で獲得 → 8万円の商品を購入。後者の方がROASが高い

### 1-2. ターゲットの理解を怠らない

- クリエイティブは最終的に「人」が見るもの
- ターゲットの生活環境、行動習慣、利用SNS、悩み、願望を具体的に理解する
- 顧客の視点からクリエイティブを設計する

### 1-3. 仮説検証から逃げない

- 最初から成功するクリエイティブは作れない
- 仮説を立て、検証を繰り返し、精度を上げていくプロセスが不可欠
- 失敗を恐れず試行錯誤を続ける

### 1-4. 訴求軸が「集まる顧客」を決める

- クリエイティブの訴求軸は、集まる人の質を根本的に決定する
- CPAが安い訴求 ≠ 良い訴求。ファネル全体（セミナー着座→個別予約→成約）で評価する
- 訴求軸ごとに「どんな人が反応するか」を事前に想像し、LTVの高い顧客を連れてくる訴求を選ぶ

#### 訴求軸別リスクパターン

| 訴求軸 | CPAへの影響 | リスク | 実例 |
|--------|-----------|--------|------|
| **プレゼント訴求**（「有料級講座あげます」） | 安い（無料に反応しやすい） | 「くれくれ君」が集まる。LINE登録後に「講座をください」で終わり、セミナー予約・成約に進まない | 個別予約率が70%→50%に低下（約29%減）。CPAが安いため予算を投下→Meta側もそこに最適化→質の低い層がさらに集まる悪循環 |
| **権威訴求**（著名人起用） | 安い（知名度で止まる） | 著名人目当てで集まり、商品・サービスへの自分ゴト化が弱い。広告学習も歪む | CPA安→予算投下→著名人ファンばかり集まり、ROASが悪化 |
| **恐怖訴求**（「このままだと危険」） | 安い（不安で反応） | ネガティブマインドの顧客が集まり、購入後のトラブル・クレームにつながりやすい | — |

**実例動画**: プレゼント訴求CR — https://www.facebook.com/reel/757621313766454
冒頭から「この有料級講座あげちゃいます」でプレゼントを前面に出した結果、くれくれ君を大量に集客。スキルプラスセミナー動線の個別予約率が70%→50%に低下した。

**共通の罠**: いずれもCPAが安いため「成果が出ている」と錯覚しやすい。しかしファネル後半の数値（予約率・成約率・LTV）を見ると利益を毀損している。

**判断基準**: 「この訴求で集まった人は、セミナーに来て、個別予約して、商品を買うか？」を常に問う。

### 1-5. 正しくお金を使う

- 無駄な広告費を避けるために知識を深める
- 一方で、広告費を抑えすぎることも機会損失になる
- トレンドに乗って集客を加速させるべきタイミングでは、思い切って投入する
- 広告は集客をコントロールできる手段。タイミングを見極めて効果的に資金を投入する

## 2. 広告集客の全体像

### 広告集客とは

- 広告を使って「見込み客になりそうな人」を狙い撃ちで集める技術
- 単に人を集めるのではなく、商品・サービスの購入につながるターゲットを集めることが目的
- 過激な表現や大衆受けするワードで人を誘導するだけでは販売には繋がらない

### 広告集客を習得する3つの理由

1. **初対面のユーザーに強くなる** -- 広告は突然表示されるもの。知らない人物・商品でもクリック・購入に誘導する技術はビジネスの基本
2. **コストとリターンの計算力が上がる** -- 広告費に対する期待値計算を繰り返すことで、施策全般（外注・ツール導入等）の判断精度が向上する
3. **様々な場面で応用できる** -- 販売導線の設計や広告以外の集客活動など、人をある地点から別の地点へ誘導する場面すべてに使える

### 見込み客を獲得する流れ

```
広告を認知 → 広告をクリック → LP（ランディングページ）に遷移 → CV（コンバージョン）
```

- 見込み客は広告を見ようと思って見ているわけではないため、まず認知させる必要がある
- ターゲットが興味を持つコピーやデザインでクリックを促す
- LPに遷移させ、無料特典・プレゼント・無料体験・セミナー予約・資料請求などでCVを獲得する
- ここまで完了して初めて「集客」が成立する

### 広告集客の4STEP

1. **広告媒体の選定** -- どこから集客するか（→ 本ファイル Section 3）
2. **オプトインLPの制作** -- どこに着地させるか（→ 本ファイル Section 6）
3. **広告クリエイティブ制作** -- 何を見せるか（→ 本ファイル Section 4, 5）
4. **広告運用** -- 数値を見て判断する。最も重要なのは数値を分析する力。分析力があれば次のアクションが自動的に決まる（→ 広告日次確認.md）

### 狙うべきターゲット層

ターゲット層は3つに分類される。最もコスト対効果が高いのは **温かい見込み客**。

| 層 | 定義 | CPA | CVR | 特徴 |
|---|------|-----|-----|------|
| 冷たい見込み客 | 問題を認識していない、または関心が薄い | 安い | 低い | 教育に時間がかかり、商品購入に繋がりにくい |
| **温かい見込み客** | 問題を理解し解決策を探している／迷っている／試して上手くいっていない | **安い** | **高い** | **ROASが最も高くなる。最優先ターゲット** |
| 熱い見込み客 | 今すぐ購入したい状態 | 高い | 高い | 母数が少なく競争が激しい。持続的な集客には不向き |

**用語**:
- **CPA** = 顧客獲得単価（Cost Per Acquisition）
- **CVR** = コンバージョン率（Conversion Rate）
- **ROAS** = 広告費用対効果（Return On Ad Spend）

## 3. 媒体選定の考え方

### 媒体選定の4つの判断基準

1. **ターゲットが存在するか** -- ターゲットのライフスタイルを把握し、そのユーザーがその媒体を利用しているか確認する
2. **ターゲティング精度が高いか** -- ターゲットが存在していても、確実に広告を届けられなければ効果は出ない
3. **配信面が安定しやすいか** -- 予算を増やしても安定して配信できるか。運用のしやすさに直結する
4. **PDCAが回しやすいか** -- エラー対応が少なく、検証を素早く実行できる媒体を選ぶ

### 主要SNS広告媒体の比較

| 媒体 | ユーザー層 | 広告形式 | ターゲティング精度 | 特徴・備考 |
|------|----------|---------|---------------|----------|
| **Meta広告**（Facebook + Instagram） | FB: 25〜54歳 / IG: 18〜34歳 | 画像、動画 | 非常に高い（豊富な顧客データ） | 初心者に最もおすすめ。バナーで配信可能、配信面安定、PDCAを回しやすい |
| **Google広告**（リスティング + YouTube） | 検索ユーザー + YouTube視聴者（幅広い年齢層） | テキスト、動画 | 高い（検索データ + 視聴データ） | 検索意図に基づくターゲティングが強み。Meta広告で数値が合わない場合の次の選択肢 |
| **X広告** | 18〜49歳（特に25〜34歳） | テキスト、画像、動画 | 低め（MetaやGoogleに劣る） | イベント告知向き |
| **LINE広告（LAP）** | 10〜60代（幅広い） | 動画、画像 | 高い（運用次第でブレあり） | オープンチャット・VOOMに配信 |
| **TikTok広告** | 18〜34歳（若年層中心） | 動画 | AI最適化が主（詳細ターゲティングは弱い） | 若年層リーチに優れる |

### 推奨媒体

- **最初に選ぶべき媒体: Meta広告** -- 上記4基準をすべて満たす。顧客データ豊富、ターゲティング精度高、バナーで低コスト配信可能、配信面安定
- **次の選択肢: Google広告** -- Meta広告で数値が合わない場合に追加。検索データに基づくターゲティングが強み
- Meta広告 + Google広告でSNS広告集客の約9割をカバーできる。年商億規模まではこの2媒体で十分
- まず1つの媒体で効果を確認し、その後集客拡大のために他の媒体を追加する

### 広告の2タイプ

| タイプ | 説明 | 例 |
|--------|------|-----|
| PULL型 | ユーザーが自ら動いて広告を見る | リスティング広告 |
| PUSH型 | ユーザーが動かなくても広告が流れてくる | YouTube広告、Meta広告 |

PULL型とPUSH型ではLPの作り方も変わるため、絶対的な正解の構成は存在しない。

## 4. 広告クリエイティブの定義と種類

### 広告クリエイティブとは

各広告媒体で配信するための制作物の総称。クリエイティブのクオリティによってROASが大きく変動する。同じ導線・同じLPでも、クリエイティブが変わるだけでROASが100〜200%変わることがある。

### 広告クリエイティブのゴール

**目的に対して「3つの壁」を越えること。**

見込み客が商品を購入するまでのステップ:

```
認知 → 興味 → 欲求 → 比較検討 → 購入
```

クリエイティブはこの流れの「入り口」を担う。クリエイティブの種類によって、認知〜興味までを担う場合もあれば、欲求の段階まで担う場合もある。

### 良い広告クリエイティブの定義

- 同じターゲット層における離脱率が低いクリエイティブ
- 同じターゲット層でCPAが安くなるクリエイティブ
- 結果としてROASが高くなるクリエイティブ

### クリエイティブの本質

**インプレッションを買って人を動かすこと。** 広告クリエイティブは人を「ある地点」から「ある地点」へ動かすためのコンテンツ。どういった見込み客をどのように動かしたいかを考えることで、良いクリエイティブが完成する。

### テキスト広告

- リスティング広告やX（旧Twitter）広告など、文字だけで出稿する形式
- テキスト広告 → LP → CV の流れが基本

### バナー広告

- 画像1枚で構成される広告形式（Meta広告など）
- **メリット**: 制作コストが低く、PDCAを回しやすい。コンセプトやコピーのテストに最適
- **役割**: 主に認知促進・興味喚起。教育力は高くないため、LPでの教育が必要
- **活用法**: コンセプトテスト → 効果があったものを動画化して強化

### 動画広告

- Meta広告、TikTok広告、YouTube広告など動画配信可能な全媒体で利用
- **メリット**: 情報量が圧倒的に多く、教育力が高い。動画内でオファーへの欲求を喚起できる
- **デメリット**: 撮影・編集にコストがかかる
- **役割**: 認知〜欲求までを1つで完結させることが可能。LPの教育力が小さくてもCVが発生しやすい

### バナー広告と動画広告の比較

| 観点 | バナー広告 | 動画広告 |
|------|----------|----------|
| 教育力 | 低い（認知〜興味まで） | 高い（認知〜欲求まで） |
| 制作コスト | 低い | 高い |
| PDCAの速度 | 速い | 遅い |
| LPへの依存度 | 高い（LP側で教育が必要） | 低い（動画内で教育可能） |
| 推奨用途 | コンセプト・コピーのテスト | CVR最大化 |

### PDCAサイクルの活用

バナー広告と動画広告を組み合わせて活用する:

1. バナー広告でコンセプト・コピーをテスト
2. 反応の良いコピー・訴求を特定
3. そのコピーを動画広告にしてCVR（コンバージョン率）を最大化

## 5. 3つの壁フレームワーク

広告クリエイティブの品質を決める最重要フレームワーク。この3つの壁を突破できるかどうかが、クリエイティブの成否を分ける。

### 第1の壁: 読まない・見ない壁

**定義**: 見込み客がクリエイティブに気づかない、または気づいても見るのをやめてしまう壁。

**重要な認識**: この壁は一度超えて終わりではなく、**超え続けるもの**。最初に興味を引くだけでなく、興味を惹き続けることが重要。

#### 突破手段（4つ）

| 手段 | 内容 | 例 |
|------|------|------|
| 文字 | ターゲットが無意識に反応する脳内ワードを使う | 業界用語、悩みに直結するキーワード |
| ビジュアル | 商品・人・世界観で興味を引く | 目を引く画像、ターゲットが共感する人物 |
| 企画 | ドッキリ・クイズ等のフォーマットで興味を維持 | 「9割が知らない〇〇」「〇〇 vs △△」 |
| 音 | 動画広告で音によって興味を引き続ける | BGM、効果音、声のトーン |

#### 訴求ワードの2分類

- **脳内ワード**: 見込み客の頭の中に既に存在しているワード（例: SNS、広告、ホットペッパービューティー）
- **興味ワード**: 多くの人が反応しやすいフックになるワード（例: 「新常識」「9割が知らない」「実は〇〇」）

### 第2の壁: 納得しない壁

**定義**: クリエイティブを見ていても、行動するための妥当性が感じられず離脱する壁。

**突破の原則**: 目的に対して行動するための説得力を、心理トリガーを用いて高める。

#### 突破手段（4つ）

**A. 信頼度を高める**

信頼には2種類ある:

1. **発信者への信頼**
   - 権威（実績・肩書き・専門性）を伝える
   - 親近感（仲の良い人、共感できる人）を演出する
   - 原則: 「何を言うか」の影響度は「誰が言うか」によって決まる
   - 人が納得するのは「偉い人」か「仲の良い人」から言われたとき

2. **オファーへの信頼**
   - 社会的証明を活用（成功事例の共有、多数の事例、レビュー）
   - 「みんなが良いと思っているもの」「みんなが成果を出しているもの」は良いものだと判断される

**B. 目的を明確にする（オファーを受け取る意味の付与）**

2つのアプローチ:

1. **ベネフィットを提示する**（プラスの目的）
   - ベネフィット = メリットがもたらすプラスの影響
   - ベネフィットは1つではなく多数存在する。その中からターゲットにとって最適なものを選ぶ
   - 例（カメラ）: 「写真が撮れる」ではなく「子どもとの大切な時間を写真に残せる」「家族との思い出を増やせる」
   - 最適なベネフィットにより「○○のために購入する」という目的が明確になる

2. **受け取らないことによる損失を伝える**（マイナスを避ける目的）
   - 人は得られる報酬よりも損失を回避したい生き物（損失回避バイアス）
   - オファーを受け取らないことでどれだけの損失があるかをイメージさせる
   - **注意**: 恐怖訴求のやり過ぎはネガティブマインドな見込み客を集めやすい。購入後のトラブルにもつながる
   - 判断基準: 「どういったお客様を集めたいか」から逆算した訴求にする

**C. 問題を意識させる**

2つのアプローチ:

1. **顕在的な問題を再認識させる**
   - 普段抱えている問題を「思い出させる瞬間」を作る
   - 方法: 感情の言語化、悩みの言語化
   - 効果: 分かりやすい言葉で共感を得る → 信頼関係の構築 → その後の話を聞いてもらいやすくなる
   - 解決策提示のティーアップにもなる

2. **潜在的な問題を顕在化させる**
   - まだ認識していない問題を掘り起こして認識させる
   - 見込み客にとって「不都合だが無視できない情報」を提示する
   - 例: 「フォロワー数と収益は関係ない」→ フォロワーを伸ばすことが常識の人にとって、自分のやり方に問題があると気づくきっかけ
   - 問題が認識できれば、解決策（オファー）提示のティーアップになる

**D. オファーを提示・強化する**

必要なこと:

1. **解決策（オファー）を提示する**
   - オファーの名前がシンプルで分かりやすいことが重要。名前が魅力的であれば数字が上がる

2. **解決策の魅力度を高める**
   - **効果に対する納得感を与える**: なぜ効くのか、具体的な根拠を示す（例: 「○○成分が配合」）
   - **従来の解決策との違いを提示する**: 新規性のあるアプローチであることを伝え、「自分の悩みを解決できるかもしれない」という感情を引き出す
   - **対立構造を作り比較する**: 何かと比べることで良い・悪いの判断を促す（例: 「動画編集は月5万まで再現性あり。プログラミングなら収益を大きく伸ばせる可能性が高い」）

### 第3の壁: 行動しない壁

**定義**: オファーの内容が良いと思っても「今受け取らなくてもいい」と後回しにする壁。

#### 突破手段

1. **限定性・希少性・緊急性を付与する**
   - 「30名限定」「先着○○名まで」「期間限定」
   - 「本気の人しか受け取らないでください」（商品への自信を打ち出す/あえて突き放す）
   - やり過ぎると逆効果。さらっと伝える程度が推奨

2. **CTAを明確に伝える**
   - 受け取りたいと思っていても、方法が分からなければ行動しない
   - 具体的な行動指示を必ず入れる
   - 例: 「下のボタンをタップして受け取ってください」「右にスワイプして次に進んでください」

## 6. LPとの一貫性（絶対ルール）

**広告クリエイティブで最も守るべきルール: LPとの一貫性を保つこと。**

一貫性がなければメッセージがブレ、見込み客はストレスを感じて離脱する。

### 保つべき一貫性は2つ

1. **広告クリエイティブ → LP** の一貫性
2. **LP → 次の目的地（LINE登録1通目等）** の一貫性

例: 広告で「食事で痩せる」と訴求しているのに、LPで「運動をしましょう」と言うのはNG。LINE登録後の1通目でも同じ訴求軸を維持する。

### オプトインLPの位置づけ

- オプトインLPは「広告クリエイティブと次の目的地をつなぐ架け橋」であり、LP単体では意味がない
- PULL型（リスティング）とPUSH型（Meta/YouTube）ではLPの作り方が変わるため、絶対的な正解の構成は存在しない

### 一貫性を保つ2つのアプローチ

#### 統合型

- 広告クリエイティブで**具体的な内容**を訴求し、LPでは**1つ抽象度が上がったテーマ**にする
- 例: LP「集客方法」→ CR「SNS集客の秘訣」「広告で新規集客2倍」「メール施策の効果」

| | 広告クリエイティブ | LP |
|--|--|--|
| 抽象度 | 低い（具体） | 高い（上位概念） |
| 例 | 「毎日10分のヨガで体が変わる」 | 「健康的な生活習慣の築き方」 |
| 例 | 「バランスの良い食事で免疫力UP」 | 「健康的な生活習慣の築き方」 |

- **メリット**: 複数の具体的な切り口でCRを展開でき、見込み客の母数を広げやすい
- **適用場面**: 規模が大きくなり、見込み客の母数を広げる必要がある段階

#### 平行型

- 広告クリエイティブとLPの**テーマの抽象度が同じ**
- 例: LP「30日で英語力を向上させる学習法」→ CR「1ヶ月で英語力がアップする効果的な学習法」「4週間で英語スキルを磨く秘訣」

| | 広告クリエイティブ | LP |
|--|--|--|
| 抽象度 | 同じ | 同じ |
| 例 | 「CPAが1000円下がった広告集客法」 | 「新規集客が2倍になった広告集客方法」 |

- **メリット**: シンプルで作りやすい
- **注意**: 抽象度が高すぎると見込み客を捉えられない（「起業」「ビジネス」は広すぎる）。狭い領域で尖らせる
- **適用場面**: 初期段階、テーマが明確な場合

### 一貫性チェックリスト

- [ ] CRとLPのテーマは同じ上位概念に属しているか
- [ ] CRで伝えた訴求内容はLPの中で回収されているか
- [ ] CRとLPの抽象度の差が大きすぎないか
- [ ] CRで触れていない別のテーマがLPに混在していないか

### NGパターン

- **メッセージのずれ**: LP「プログラミングでキャリアアップ」なのにCR「簡単な副業で月10万円」
- **抽象度の乖離**: LP「効果的なマーケティング戦略」なのにCR「Instagram投稿の最適な時間帯」「SEO対策のメタタグの書き方」

## 7. 広告審査

### 審査の流れ

```
広告入稿 → AI審査（画像・テキスト・見出し・LP） → 配信開始 → 人による審査 → 最終判定
```

- AI審査を通過しても、後から人の審査で広告がストップすることがある
- 人の審査のほうが厳しく、媒体ポリシーに反するものは停止される

### 審査に落ちやすい広告の3パターン

#### 1. 非現実的な結果を想起させる表現

| NG表現 | 理由 |
|--------|------|
| 「必ず稼げる」「必ず痩せる」 | 結果を断定している |
| 「最短」「最速」「最高」 | 最上級表現 |
| 「3ヶ月で月商○○万円」 | 期間と成果がセットになっている |

#### 2. NGワードを含む表現

- NGワード例: 「月収」「稼ぐ」「月利」「不労所得」
- 略語もNG: 「Facebook」を「FB」と略す等
- 絵文字の多用、文字の大小の間違いもアカウント停止リスクあり

#### 3. 不適切な文言や画像

| NG表現 | 改善例 |
|--------|--------|
| 「24歳のあなたに」 | 「若年層向けのサービス」（個人を特定しない） |
| 「うつでお困りのあなたに」 | 「うつ病のカウンセリングを受けられます」（断定しない） |
| 過度な露出のある画像 | 適切な範囲に抑える |
| 極端なビフォーアフター画像（腹筋アップ、ぶかぶかズボン等） | 控えめな変化表現にする |

- 商標登録されている文言の使用は基本的に禁止

### 審査突破のルール

- **最重要: 広告媒体のポリシーを守る**
- 媒体側の立場で考える -- ユーザーが離脱しないよう、ネガティブな広告は流したくない
- 短期的にAI審査を通過しても、ポリシー違反はアカウント停止・永久BAN のリスクがある
- 広告を出す前に、一度媒体のポリシーに目を通しておく

## 8. チェックリスト

### 企画段階

- [ ] ゴール（目的）は明確か
- [ ] ターゲットは具体的に定義できているか
- [ ] ROASの観点で評価する準備ができているか

### 3つの壁の突破設計

- [ ] 第1の壁（読まない・見ない）: 興味を引き続ける仕掛けがあるか（文字/ビジュアル/企画/音）
- [ ] 第2の壁（納得しない）: 信頼・目的・問題意識・オファーの4要素が入っているか
- [ ] 第3の壁（行動しない）: 限定性とCTAが明確か

### 品質チェック

- [ ] LPとの一貫性は保たれているか（統合型 or 平行型）
- [ ] ターゲットの脳内ワード・興味ワードが使われているか
- [ ] ベネフィットが「メリット」ではなく「メリットがもたらすプラスの影響」で表現されているか
- [ ] 恐怖訴求が過度になっていないか
- [ ] オファーの名前はシンプルで分かりやすいか

### 審査チェック

- [ ] 媒体のポリシーを一読したか
- [ ] 結果を断定する表現を使っていないか
- [ ] 最上級表現（最短・最速・最高）を使っていないか
- [ ] NGワード（月収・稼ぐ・月利・不労所得）を含んでいないか
- [ ] 略語・絵文字の乱用をしていないか
- [ ] 個人を特定する断言的な表現を使っていないか
- [ ] 画像に過度な露出や極端なビフォーアフターがないか
- [ ] 商標登録された文言を使っていないか
