# Meta広告アカウントエラー対応

## 役割

Meta広告で発生する `広告却下 / Facebookページエラー / Instagram連携エラー / 広告アカウント制限 / ビジネスマネージャ制限 / Facebook個人アカウント停止` に対して、相談者が今やるべき行動を最短で提示する。

## ゴール

- エラーを `いま何をすれば良いか` に変換する
- 本文は短く保ち、詳細は資料へ逃がす
- Meta側の誤認と、実際のポリシー違反を混同しない
- 不正な迂回を提案しない

## 大前提

- Meta広告はポリシー準拠が前提
- 作りたてアカウントでは AI 判定で制限されることがある
- ただし誤認のケースも多い
- エラーが出たら、放置せず `修正 / 審査リクエスト / 本人確認` を迅速に進める

## まず切る分類

### 1. 設定エラー

- 症状
  - 管理画面の右上や広告セット画面で設定不足のエラーが出る
- 今やること
  - 画面の指示に従って修正する
- 補足
  - これはポリシー問題ではなく設定不足のことが多い

### 2. 作りたてアカウントの制限

- 症状
  - 新規作成直後に広告やアカウントが制限される
  - Instagram連携時に制限表示が出る
- 今やること
  - `審査リクエスト` を出す
  - `本人確認` が出たらそのまま進める
- 補足
  - AI誤認のケースが多い
  - まずは解除導線を踏む

### 3. 広告却下

- 症状
  - 広告が却下される
  - 広告が配信されない
- 今やること
  - テキスト、画像、LPを見直す
  - 誤判定っぽければ `アカウントのクオリティ` から再審査
- よく当たる論点
  - `個人的特質`
  - `誤解を招く主張`
  - `システムの回避`
  - `機能しないLP`
  - `広告とLPの不整合`

### 4. セキュリティ停止

- 症状
  - 不正利用や本人性に関する制限
  - 普段と違う場所や端末からのログイン後に停止
- 今やること
  - 本人確認
  - サポート連絡
- 補足
  - 複数人で 1 つの個人アカウントを共有していると止まりやすい

### 5. 長期未使用停止

- 症状
  - 60日以上利用履歴が少ない、または停止後に長期間放置
- 今やること
  - 再審査
  - サポート連絡

## よく引っかかる原因

### 個人的特質

- NG になりやすい例
  - `〜に悩んでいるあなた`
  - `18〜24歳の方向け`
- 方向性
  - ターゲットの属性を断定せず、サービス説明に寄せる

### 誤解を招く主張

- NG になりやすい例
  - `必ず稼げます`
  - `3ヶ月で売上500万`
  - `6ヶ月でマイナス10kg`

### 迂回と見なされる表現

- NG になりやすい例
  - 絵文字や Unicode 記号の多用
  - 認証ドメインと遷移先ドメインの不一致

### LPの問題

- NG になりやすい例
  - 情報が薄い
  - 離脱阻害ポップアップ
  - 広告文とLP訴求のズレ

## 返信文の原則

- 先に `今やること`
- 次に `なぜその対応か`
- 最後に `詳しくはここ`
- 本文に全原因を書ききらない

## representative response

### 作りたてアカウント制限

```text
お疲れ様です！
甲原です！
Instagramアカウントが制限された時は、まず「審査リクエスト」を出して、本人確認まで進めてください！
アカウントを作りたてのときは、AIの判断でアカウント制限になることがありますが、誤認のケースも多いです。
なので、放置せずにその場で審査リクエストと本人確認まで進めるのが大事です！

詳しくは下のマインドマップに書いていますので、ご確認ください！
https://mm.tt/app/map/2908015838?t=0W9RgdFZoW
```

### 広告却下

```text
お疲れ様です！
この場合は、先に広告文、画像、LPのどこで引っかかっているかを見て、修正できる箇所を直してください！
もし明らかに問題が見当たらないなら、「アカウントのクオリティ」から再審査を出して大丈夫です！

Metaは個人的特質、誇張表現、LP不整合あたりで落ちることが多いので、そのあたりを中心に見ると切り分けしやすいです！
```

## サポート問い合わせ

- 入口
  - 広告マネージャー左下の `?`
  - `サポートチームに連絡`
- 方針
  - チャットがあればチャット優先
  - なければメール
- 注意
  - 返答は数日以上かかることがある
  - 何度も連続で催促しても早まるとは限らない

## 復活しない場合

- まず確認
  - 再審査をやったか
  - サポート連絡をやったか
- それでも不可
  - 新規ビジネスマネージャーまたは個人アカウント作成を検討する
- ただし
  - 先に飛ばない
  - 不正利用と見える再作成を勧めない
