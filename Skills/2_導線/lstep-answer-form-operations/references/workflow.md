- `回答フォーム`
- `新しい回答フォーム`

代表 route
- 新規: `/lvf/edit/new`
- 既存: `/lvf/edit/{form_id}`

- `フォームタイトル`

- `保存`

  - `スプレッドシート連携`
  - `回答状態`

## exact な最小作成手順

1. `回答フォーム`
2. `新しい回答フォーム`
3. `フォーム名(管理用)`
   - current input name は `formname`
4. `タイトル`
   - current input name は `pagetitle`
5. `記述式(テキストボックス)`
6. 質問名を入れる
7. `保存`

## representative の読み方

### アンケート型

- 何を感じたか
- 何に困っているか
- 次に何を望むか
を回収する

### 申込前質問型

- 申込前の温度感
- 現状
- 適合性
を回収する

### 回答後分岐型

- 回答内容をもとに次の message や営業対応を変える

### current 用途の代表

- `NPS`
  - 満足度や推奨度を取る
- `ゴール設定`
  - 受講後の行動や目標を取る
- `合宿`
  - 参加情報や当日運用情報を取る
- `数値管理`
  - 運用上の structured data を取る
- `アカウント突合フォーム`
  - account や受講状況の突合に使う

## 30秒レビューの順番

1. `回答フォーム`
2. 一覧の `フォーム名`
3. `スプレッドシート連携`
4. `回答状態`
5. `公開状態`
6. 何を回収するフォームか
7. 回答後に何を変えるか
8. `... -> 詳細を確認` または edit 画面で質問種別を見る
9. 必要なら `... -> テスト送信`
10. 実機で開いて、回答者目線で迷いがないか確認

## current live で固定したこと

- create route は `/lvf/edit/new`
- visible 主入力は
  - `input[name="formname"]`
  - `input[name="pagetitle"]`
- `formNameInput` / `formTitleInput` のような抽象呼び方ではなく、現在は visible field 名で読む方がズレにくい
- 最初の質問 block は `記述式(テキストボックス)`
- save 後は `/lvf/edit/{form_id}` に残る
- 一覧 row menu は
  - `詳細を確認`
  - `公開設定を変更`
  - `コピー`
  - `テスト送信`
  - `削除`

## 保存前の最小チェック

- 何を回収したいフォームか 1 文で言える
- その回答を使って次に何を変えるか言える
- `記述式(テキストボックス)` / `段落(テキストエリア)` / `ラジオボタン` / `プルダウン` の選択理由がある
- `スプレッドシート連携` が必要か不要か判断できている
- `サンクスページURL` と `回答後アクション` の主従が切れている

## 保存後の最小チェック

- 一覧に戻って `フォーム名` が想定どおりに見える
- `公開状態` が意図と合っている
- 必要なら `... -> テスト送信`
- 実機で開いて、質問順と文言が自然か確認する
- テスト用なら `... -> 削除 -> 削除する` まで行う
- `サンクスページURL` があるなら到達先を押して確認する

## ここで止めて確認する条件

- 回答後に `どのタグ / シナリオ / 営業判断` へつなぐか決まっていない
- `スプレッドシート連携` を新規で使う
- 回答内容で分岐を変えたいが、既存フォーム流用で足りるか曖昧
- current フォームを編集するか、新規フォームを作るか迷う
- `自由記述` にしたい理由が `何となく詳しく知りたい` 以上に下りない
- `サンクスページURL` と `回答後アクション` のどちらが主役か言えない

## 完成条件

- 何を回収したいフォームかを 1 文で言える
- `スプレッドシート連携 / 回答状態 / 公開状態` の読み方を説明できる
- 実機で開いて質問順と文言を確認している
- 回答後に何を変えるフォームか説明できる
- テスト用なら削除まで終えている
- 回答後の `見える体験` と `裏の state 変化` を分けて説明できる

## 質問種別を切る時の基本

- 自由記述が必要
  - `記述式(テキストボックス)` または `段落(テキストエリア)`
- 単一選択でよい
  - `ラジオボタン` または `プルダウン`
- 複数選択させたい
  - `チェックボックス`
- 添付が必要
  - `ファイル添付`
