# Lステップ アクション管理 exact メモ

## current 入口

- `アクション管理`
- `新しいアクション`

代表 URL
- 一覧: `/line/action`

## current で見えている主要ラベル

- `新しいフォルダ`
- `新しいアクション`
- `並び替え`
- `アクション名`
- `登録日`
- `スケジュール設定`

## current で見えている動作種別

- `テキスト送信`
- `テンプレート送信`
- `タグ操作`
- `友だち情報操作`
- `シナリオ操作`
- `メニュー操作`
- `リマインダ操作`
- `対応マーク・表示操作`
- `イベント予約操作`
- `コンバージョン操作`

## 最小の exact 作成手順

1. `新しいアクション`
2. `テキスト送信`
3. `保存するアクション名`
4. `1. テキスト送信`
5. `メッセージ`
6. visible `[contenteditable="true"]` に本文
7. `この条件で決定する`

## reusable の代表的な読み方

### 状態更新型

- タグ解除
- タグ追加
- 対応マーク変更

### 送信連動型

- テンプレート送信
- テキスト送信

### 制御型

- シナリオ停止
- メニュー変更
- イベント予約操作

### 複合 macro 型

- 例: `アップセル者対応`
- 読み方
  - タグ解除
  - タグ追加
  - テンプレ送信
  - シナリオ停止
  - 対応マーク変更
  を 1 action に束ねる

## current で未完了の selector 系

- `タグ操作 -> タグ選択`
- `テンプレート送信 -> テンプレート選択`
- `イベント予約操作 -> 予約枠選択`

これらは current representative を見ながら使う。最小 skill の正本は `テキスト送信` を基準にする。
