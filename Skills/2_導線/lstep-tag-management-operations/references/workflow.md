# Lステップ タグ管理 workflow

## current で固定している事実

- 一覧 URL
  - `https://manager.linestep.net/line/tag`
- 上部 button
  - `新しいフォルダ`
  - `新しいタグ`
  - `CSVアップロード`

## live で確認した exact 手順

1. `タグ管理`
2. folder を開く
3. `新しいタグ`
4. dialog title
   - `タグの新規作成`
5. field
   - `タグ名`
6. button
   - `キャンセル / 作成`
7. 一覧 row 右端 `more_vert -> 削除`

## 読み方

- visible 一覧が空でも、folder 件数や action 側で実際に使われていることがある
- つまり `タグ一覧が少ない = 何もしていない` とは見ない
