# オプトインLP制作 5STEPフロー

## このスキルについて

オプトインLP（ランディングページ）を制作するための体系的な5STEPフロー。リサーチからデザインまで、CVR（コンバージョン率）の高いLPを作るための知識・フレームワーク・チェックリストをまとめたナレッジ。

---

## 前提知識: オプトインLPとは

### 定義

オプトインLPとは「広告クリエイティブと次の目的地を繋ぐもの」。LINE登録やメール登録後の流れを踏んでもらうためのティーアップとして機能する。

### なぜLPが必要か

- マーケティングはエモーショナルリレー（感情の流れ）が重要
- LPがないと「広告CR → 登録 → メッセージ」の展開が急すぎ、一部の層しか獲得できずCRもすぐ枯れる
- LPがあることで見込み客の登録直後の状態が「発信を受け取る準備が整った状態」になる
- 感情の流れをシームレスに組むことが重要

### 最低限の基本構成（4要素）

1. **ヘッドライン** -- キャッチコピー。ターゲットとベネフィットを明確にする
2. **オファー** -- 取引の条件。「何がもらえるのか」を魅力的なタイトルで見せる
3. **提供者情報** -- 権威性を出すための実績、発信の理念（簡潔に）
4. **CTA** -- とにかく目立たせる。アクションベースのコピー（「無料で参加する」「今すぐ見る」等）

### LP制作で守るべき2つのルール

1. **LP単体だけで見ない** -- 前後の繋がり（一貫性）がないと離脱が増える。LPは全体の一部分
2. **見てイメージできるようにする** -- 見込み客はLPを「読む」のではなく「見て」いる。流し見の中で気になるワードに反応してもらう設計が必要。図解・画像で視覚的にイメージさせるとCVRが上がる

### LPを定数にするべき理由

広告マーケティングでは「変数を固定する（動かさない箇所を決める）」ことが重要。

**優先順位（後ろほど動かさない）:**
1. 広告CRでPDCAを回す（変数）
2. 良いLPを制作（定数）
3. 良い導線を構築（定数）
4. 良い商品を作る（定数）

目標は「広告CRのみでPDCAを回す体制」を整えること。

---

## CVRの決まり方

```
LPの魅力度 = コンセプトの良さ x 伝える技術
伝える技術 = ライティング x デザイン
```

- CVする人数は魅力度に応じて**指数関数的**に伸びる
- 40点のLPと80点のLPは点数2倍だが、CVRは10倍以上変わりうる
- コンセプトが弱ければ伝える技術だけではCVRを大きく改善できない
- 良いコンセプトを作った上で、効果的なライティング・デザインで伝える

## STEP 1: リサーチ

### リサーチのゴール

効果的なLP制作の土台を作ること。以下の**5つの質問**に答えられる状態にする。

1. ペルソナが満たしたい需要は何か？
2. ペルソナが反応しやすいワードは何か？
3. 競合LPの特徴は何か？
4. 競合が提供する解決策は何か？
5. 自社商品独自の解決策は何か？

この5つの質問の答えがクリティカルになるほど、後続のコンセプトメイク・構成決め・ライティング・デザインすべてが良くなる。

### 効果的なリサーチ方法（7種）

#### 1. 既存顧客インタビュー

実際の顧客と話すことでリアルな言葉や需要が得られる。以下を必ず聞く。

| 質問 | 目的 |
|------|------|
| 商品を購入する前の状態や悩みは何か？ | 需要の把握、ペルソナ情報の取得 |
| なぜ悩みを自分で解決できなかったのか？ | 満たされていない深い需要の発見 |
| 購入前に悩みを解決する行動はしたか？ | 競合の解決策の特定 |
| 商品購入前の目標は何か？ | ベネフィットのレベル感の把握 |
| 商品を購入する決め手は何か？ | 商品力の確認（「あなただから」は危険信号） |
| 価格以外で買わない理由はあったか？ | 商品改善のヒント |
| 商品の価格についてどう思ったか？ | 魅力の伝わり方の確認 |
| 購入してどんな未来が見えているか？ | ビフォーアフターの変化 |
| なぜ他社ではなく自社を選んだか？ | 独自性のポイント |
| この商品がなかったらどうしていたか？ | 代替品・競合の把握 |
| この商品を紹介するなら、どう紹介するか？ | 顧客が感じた本当の魅力 |
| もっとあれば良いと思う商品はあるか？ | 未充足の需要 |

#### 2. キーワードリサーチ

- 商品ジャンルを決め、関連キーワードを洗い出す
- **ラッコキーワード**でサジェストキーワード（実際に検索されているワード）を取得
- **Googleキーワードプランナー**で検索ボリュームを確認
- 理想は月間検索ボリューム **1,000〜10,000** の範囲（少なすぎると獲得困難、多すぎると競争過多）

#### 3. 競合リサーチ

- **Meta広告ライブラリ**（無料）で競合の広告・LP・訴求を調査
- 長期間掲載されている広告は当たっている可能性が高い
- クリエイティブとLPのパターンを保存して分析

#### 4. SNSリサーチ

- YouTube、X（旧Twitter）、Instagram、TikTokで調査
- 伸びている投稿やエンゲージメントの高い投稿を分析
- 再生回数・コメント欄から需要や反応ワードを把握

#### 5. 広告リサーチ

SNSリサーチとほぼ同じ流れ。特に広告として配信されているコンテンツの訴求パターンを収集。

#### 6. 悩みリサーチ

- Yahoo!知恵袋やQ&Aサイトでリアルな悩みを調査
- 具体的な言葉がそのままLP訴求に使える

#### 7. レビューリサーチ

- Amazonレビューやコンテンツ販売サイトのレビュー
- ポジティブ意見 → 訴求に活用
- ネガティブ意見 → 改善点・差別化ポイントの発見

## STEP 2: コンセプトメイク

### メインコンセプトとサブコンセプト

| 項目 | メインコンセプト | サブコンセプト |
|------|----------------|--------------|
| 数 | 1つ | 複数 |
| 定義 | 商品全体を通して伝えたいメッセージの中心軸 | メインコンセプトを具体化した要素 |
| 抽象度 | 高い（全体の方向性） | 低い（尖ったメッセージ） |
| 決めるタイミング | 販売導線構築時 | LP制作時 |

- メインコンセプトが最初から狭く尖っていれば、サブコンセプトと同じ抽象度になることもある
- サブコンセプトは必ずメインコンセプトに内包されていること（一貫性が重要）
- 一貫性がないと「思ったのと違う」という離脱が発生する

### LPのコンセプト: 2つのパターン

1. **メインコンセプトと同じもの** -- 販売導線構築・商品設計のコンセプトをそのまま使う
2. **メインコンセプトより具体性が高いもの（サブコンセプト）** -- ターゲットが欲しいと思うような具体的コンセプトに変える

### 反応率が高いLPのコンセプト特徴

- **競合の上位互換** -- 既存の解決策を超える
- **競合がいないポジション** -- 独自の切り口
- **トレンドとの掛け合わせ** -- 時流を取り込む

### 良いコンセプトの5つの要素

1. **シンプルで分かりやすい** -- ターゲットが分かる言葉で提示。イメージできないのは何も言っていないのと同じ
2. **ターゲット層が明確でインサイトを的確に捉えている** -- 自分に関係のあるワードが入っており、ベネフィットもターゲットのインサイトから導かれたものである
3. **独自性があってターゲット層に新規性がある** -- 過去に試して失敗した解決策と同じでは聞いてもらえない
4. **記憶に残るほどの意外性がある** -- 痛みをダイレクトに解消できるかもしれない、思い込みの蓋が外れる。脳内SEO（優先順位）が上がる
5. **言葉から共感を生むストーリーが想起される** -- 過去のつらい経験や痛みが思い出されるワード

### 売れるサブコンセプトを作る3STEP

1. **ターゲットのインサイトを明確にする** -- インサイトは複数存在する。メインコンセプトより具体的な部分にフォーカス
2. **独自性や意外性のある悩みの解決方法を決める**
3. **解決方法のベネフィットを付け足す**

コンセプトの構成要素: **ターゲット x 解決策 x ベネフィット** -- リサーチで見つけたワードをうまく掛け合わせる。

### コンセプト事例

| 商品 | コンセプト | ターゲットワード | ベネフィットワード | 解決策 |
|------|----------|--------------|--------------|------|
| YOLUシャンプー | 忙しくて夜の美容タイムが取れない女性に、夜のケアで朝の美髪を実現する持続型シャンプー | ケア、髪、シャンプー | ゆとりのある朝、うるつや | 夜間美容、ナイトケア、持続型シャンプー |
| ピットソール | 痩せたい人に骨格筋量を増やして美しい姿勢に矯正することで痩せる体に変えるインソール | 体型、痩せたい | 美しい姿勢、痩せる体 | インソール（ターゲットの頭にない新しい解決策） |
| SHE likes | 私らしい働き方に出会える女性向けキャリアスクール（サブ: 在宅でSNS動画クリエイターになりませんか？等） | 副業、働き方 | 好きと得意を仕事に | 幅広いスキル学習 |

## STEP 3: 構成決め

### 構成とは

LPの目的達成のために必要な要素の配置を決めること。

### オプトインLPを構成する要素

| 要素 | 役割 |
|------|------|
| **ファーストビュー（FV）** | キャッチコピー・サブコピー・オファーの価値を伝える。最も数値に影響する |
| **オファー** | FVで伝えきれなかったオファーの価値を具体的に伝える（例: 選ばれる5つの理由） |
| **ベネフィット** | オファーを受け取った時の理想の状態を具体的に伝え、欲しい度合いを上げる |
| **社会的証明** | 他人の成功事例でオファーの価値と信頼性を高める（お客様の声等） |
| **ビジュアル** | 視覚的に訴求する必要がある商品向け |
| **緊急性・希少性** | 「今行動しないとダメ」という感情にさせる（例: 予約殺到、お早めに） |
| **信頼性向上** | 発信者の実績提示、理念で不安を取り除く |
| **Q&A** | 見込み客の不安を解消してCVRを高める |
| **フッター** | 利用規約・プライバシーポリシーのリンク |
| **CTA** | CVに必要なボタン（「無料で見る」「今すぐ受け取る」等） |
| **ポップアップ** | 離脱しようとしている見込み客に対して表示 |

### 構成の流れ（概念）

注目を集める → 価値を伝える → 信頼性を構築 → 行動を促す → 不安を取り除く

### 効果的な構成決めの5STEPプロセス

1. **コンセプトの再確認** -- メッセージがずれないよう一貫性を保つ
2. **LPに必要な要素を洗い出す** -- 競合他社が入れている要素も含めて全て洗い出す
3. **洗い出した要素を配置する** -- 競合の構成トレースも最初はOK
4. **各要素の大まかな役割と目的を定義する** -- ブロックごとに見込み客のビフォーアフター（入る前と出る時の状態）を定義
5. **全体の流れを確認する** -- ビフォーアフターの繋がりが適切か検証

### ターゲット温度感による構成の違い

- **冷たい層**: 冒頭で問題提起が必要
- **熱い層**: すぐに社会的証明（お客様の声）を上の方に配置

### よくあるLP構成パターン

**パターンA:**
FV → お客様の実績 → 自分の実績 → オファーのベネフィット → オファーの内容 → 特典 → 登録までの流れ → プロフィール → CTA（2〜3箇所）

**パターンB:**
FV → オファー → お客様の声 → プロフィール

**パターンC:**
FV → 問題提起 → オファーのベネフィット → 自分の実績 → お客様の声 → 登録までの流れ → Q&A

## STEP 4: LPライティング

### ライティングのゴール

コンセプトの価値を言葉で最大限に伝えること。読んだらCVする、流し見でもCVする状態を目指す。

### 超えるべき3つの壁

1. **読まない・見ない壁**
2. **納得しない壁**
3. **行動しない壁**

### 魅力的なLPライティングを実現する6要素

| # | やるべきこと | 目的 |
|---|----------|------|
| 1 | 見込み客に反応し続けてもらう | 読まない壁を越え続ける |
| 2 | 提供者やオファーに対して信頼してもらう | 信頼の土台を作る |
| 3 | 欲求を掻き立てて目的を明確にする | 願望・恐怖による動機付け |
| 4 | 目的達成できていない自分の問題を認識してもらう | 問題の顕在化 |
| 5 | 提供するオファーを提示して強化する | 解決策の魅力を伝える |
| 6 | コスト感を取り除く | 感情的・金銭的・時間的不安の解消 |

### 人を動かすライティング技術20

#### カテゴリ1: 反応し続けてもらう

| # | テクニック | 説明 |
|---|---------|------|
| 1 | **無意識に反応する脳内ワードを使う** | ターゲットが興味・関連する言葉をLP中に散りばめる |

#### カテゴリ2: 信頼してもらう

| # | テクニック | 説明 |
|---|---------|------|
| 2 | **権威を与えて説得力を増す** | 自分の実績をしっかり伝える。「誰が言うか」は「何を言うか」より影響が大きい |
| 3 | **権威を借りて信頼度を上げる** | 「○○に掲載」「○○から紹介」で信頼を借りる |
| 4 | **理念を伝えて価値観を共有する** | 良い理念 + 実績で信頼度が上がる。言行一致が前提 |
| 5 | **一般的な条件でも実現可能だと伝える** | 特殊条件ではなく普通の人が達成できた事例を見せる |
| 6 | **事例の多さで再現性を担保する** | 多くの成功事例がオファーへの信頼を高める |
| 7 | **手間訴求によって解決策の信頼度を上げる** | 「何百種類もの実験をして見つけた」等、かかった労力を伝える |

#### カテゴリ3: 欲求を掻き立てて目的を明確にする

| # | テクニック | 説明 |
|---|---------|------|
| 8 | **生命に関わる一次的欲求に対して訴求する** | お金・健康・仕事・恋愛・美容など、生存や子孫に紐づく訴求は必ず入れる |
| 9 | **理想の未来をイメージさせオファーを受け取る意義を付与する** | ターゲットがイメージできる適切なベネフィットを伝える |
| 10 | **損失可能性を明示して行動するべき納得感を与える** | 恐怖訴求。人は得るより失う恐怖のほうが強い。今行動しないことによる未来の損失を示す |

#### カテゴリ4: 問題を認識してもらう

| # | テクニック | 説明 |
|---|---------|------|
| 11 | **感情を言語化して問題を認識させる** | 見込み客の感情を言葉にして自分事化させる（「こんなこと思ったことありませんか？」） |
| 12 | **潜在的な問題にアプローチして問題を顕在化する** | まだ認識していない問題を気づかせる（例: ビフォーアフター写真で損失に気づかせる） |

#### カテゴリ5: オファーを提示・強化する

| # | テクニック | 説明 |
|---|---------|------|
| 13 | **問題を解決するためのオファーを提示する** | 分かりやすくイメージできる形で解決策を伝える |
| 14 | **解決策の効果に対する納得感を与える** | 「選ばれる4つの理由」等、効果が出る仕組みを伝える |
| 15 | **従来の解決策とは違う特徴を明示する** | 競合との違い、独自性を伝えて選ばれる理由を作る |
| 16 | **従来の解決策との対立構造を作って比較する** | 「独学3年 vs プロから学ぶ1年」のように比較対象を設けて良さを引き立てる |
| 17 | **特典を用意してお得感を演出する** | 付加価値として打ち出す（全面には出さない）。限定性・緊急性とセットが効果的 |
| 18 | **限定性や緊急性を伝えて行動可能性を上げる** | 「30名限定」「先着」「期間限定」等。人は「今じゃなくてもいい」で行動しない |

#### カテゴリ6: コスト感を取り除く

| # | テクニック | 説明 |
|---|---------|------|
| 19 | **コスト感を除去して安心感を与える** | 感情的・金銭的・時間的不安を全て潰す |
| 20 | **オファーの受け取り手順をイメージできるように明示する** | 受け取り方が分からないとストレスになり離脱する |

## STEP 5: LPデザイン

### LPデザインとは

コンセプトの価値を視覚的に最大限伝えること。どの文字を強調し、どういった順番で視線を誘導し、視覚的にどう印象を付けるかを設計する。

### デザインの6つの構成要素

| 要素 | 役割 |
|------|------|
| **余白** | 要素間の隙間。読みやすさ・開放感の演出 |
| **フォント** | 文字の形。細い/太いで印象が変わる |
| **カラー** | 色による印象の変化 |
| **視線誘導** | ユーザーが文字を見る順番をコントロール |
| **レイアウト** | デザイン要素の効果的な配置。情報の分かりやすさ |
| **あしらい** | 装飾的な要素。デザインの個性と世界観の構築 |

### LPデザイン制作 5STEPフロー

#### STEP 5-1: アイデア出し

- 発散ベースで世界観のアイデアを洗い出す
- ターゲットや目的によって世界観は様々（例: パーソナルジムでもマッチョ向け / 女性向けヨガ / 初心者向け等）
- できるだけ多く洗い出してからコンセプトに合うものを絞る

#### STEP 5-2: 世界観構築

- アイデアを1つに絞り、制作のブレをなくす
- 世界観が決まれば具体的なデザイン要素（カラー・フォント・レイアウト・素材）が定まる
- 例: 「海辺のリラックスカフェ」→ キーワード: リラックス/爽やか/自然/開放感 → カラー: 青/白/ターコイズ → フォント: セリフ体 → レイアウト: 余白多め → 素材: 水彩画風/丸みのあるアイコン

#### STEP 5-3: リサーチ

- 世界観から具体的なカラー・あしらい・フォントを決める
- 競合のLPやPinterestで世界観をリサーチ
- AIを活用して「このカラーはどういった印象を与えるか？」等を確認
- 個人の好みではなく、コンセプトに合うものを選ぶ

#### STEP 5-4: ラフ作成

- 情報を分類分けする（タイトル / サブタイトル / CTA等）
- 各ブロックで伝えたい情報の優先順位を決める
- 視線誘導を設計する（10人いれば10人が同じ順番で見るようにコントロール）
- 免責文を目立たせる等のミスを防ぐ

#### STEP 5-5: 実装

- ラフの通りにレイアウトを行う

## LP改善・CVR向上テクニック

### LP改善の基本

- KPIをしっかり設定し、達成できていれば次に進む
- CVRが低い原因がLPなのか、広告CRなのか、訴求方法なのかを吟味すること

### 重点的な改善箇所（数値変化が大きい3箇所）

| 箇所 | テスト方法 |
|------|----------|
| **FV（ファーストビュー）** | サブコンセプト・脳内ワードをテスト。うまくいったものをブラッシュアップ |
| **オファー** | (1) 価格を下げてみる（CVR変わらなければ前段階に問題あり） (2) 名前を変える（例: 無料相談 → ロードマップ作成会） (3) 限定性を強める（100名限定、先着等） |
| **申込フォーム** | 簡単にすること。入力項目を減らす、選択式にする |

### CVRを上げるテクニック5選

#### FV周り

- [ ] ターゲットとベネフィットが明確か
- [ ] 脳内ワードが目に飛び込むデザインか
- [ ] ターゲットとFVに一貫性があるか

#### オファー周り

- [ ] 魅力的な名前になっているか
- [ ] 価格の前段階に問題はないか
- [ ] 限定性の見せ方に工夫はあるか

#### CTA周り

- [ ] わかりやすいか
- [ ] マイクロコピーが入っているか（ボタンの下に「無料トライアル」等）
- [ ] 申込のハードルは低いか

#### 各項目の見出し

- [ ] 脳内ワードが入っているか
- [ ] 脳内ワードが目につくデザインか

#### その他

- [ ] 解決策がイメージしやすいか
- [ ] 解決策が魅力的か

## ヘッドラインの作り方

### 基本構造

**ターゲットワード x ベネフィットワード** の掛け合わせ。

- **ターゲットワード**: 無意識に反応するワード・フレーズ（「○○のあなたへ」だけではない。脳内ワード）
- **ベネフィットワード**: ターゲットの理想状態をイメージでき「それいいな」と思ってもらえるもの。具体的であるほど良い

### ヘッドライン事例

- 「SNSで億超えを達成した本質のSNSマーケ」
- 「CPAを1,000円台で獲得するMeta広告運用マニュアル」
- 「高単価デザイナーになる5STEP」
- 「足首-2cmで自信に満ちた毎日を」

## オファーの見せ方

### オファーの種類

よく使われるフォーマット: 「○○の方法」「○○メソッド」「○○マニュアル」「○○ステップ」

### 2つのパターン

1. **キャッチコピーとオファーがセット** -- 例: 「CPAを1,000円台で獲得するMeta広告運用マニュアル」
2. **キャッチコピーとオファーが別々** -- 抽象的なオファーを分解して具体化。例: 「Meta広告運用マニュアル」→「CPAを1,000円下げる運用方法」「広告CRを長持ちさせる運用方法」「アカウントバン対策のための運用」

### オファーの鉄則

- 具体的なほうがイメージしやすくCVRが高くなる
- 抽象的なオファータイトルは避ける
- タイトルが命

## CTAのベストプラクティス

- **目立たせる**: ボタンを動かす、メインカラーと対比する色を使う
- **アクションベースのコピー**: 「無料で参加する」「無料で見る」「今すぐ見る」
- **マイクロコピー**: ボタンの近くに補足テキスト（「無料トライアル」等）
- **位置**: LP内に2〜3箇所配置
