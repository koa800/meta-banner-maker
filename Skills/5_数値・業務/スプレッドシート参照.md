- **商品マスタの主なカラム**: 商品ID, 商品管理コード, 商品名, 事業区分, 対象顧客, 顧客属性区分, 価格, 購入形態, 初期費用, 商品種類

  - `商品マスタ`: 正式商品名の正本。`商品ID` を主キーにし、`商品管理コード` は分類が確定した商品のみに付与する。更新時は `商品マスタ` 自体を正本にしつつ、分類ロジックだけを上から当てる
  - `商品区分たたき台`: `商品ID / 商品管理コード / 商品名 / 事業区分 / 対象顧客 / 顧客属性区分 / メモ` だけを見るレビュー用タブ
  - `決済商品変換マスタ`: 決済CSVの生商品名を正式商品名へ寄せるための変換ルール。`要確認 / 商品マスタ未登録 / 不明` の行は、判断対象セルが色付きで残る
- **補足**:
  - `Live授業`、`みかみとお茶会`、`スキルプラスイベント`、`センサーズ　継続`、`AIカレッジ　継続` は正式商品として `商品マスタ` に追加済み
  - `AIコンテンツ自動量産Live参加費用 -> Live授業`、`VIPチケット（先着5名） / 一般チケット / 両方参加 -> スキルプラスイベント`、`センサーズ【24分割プラン】※頭金10万円銀行振込用 / 新生センサーズ限定プラン【頭金10万円＋24分割】 -> センサーズ　継続`、`AIカレッジ【24分割プラン】※頭金10万円銀行振込用 / AIカレッジ【頭金10万円＋24分割払いプラン】 -> AIカレッジ　継続` のような明示エイリアスは `決済商品変換マスタ` で自動寄せする
  - `決済商品変換マスタ` では、明示名で確定しない未確定行に限り `参照金額 ÷ 参照件数 = 798,000` を `スキルプラススタンダードプラン` とみなして自動寄せする
