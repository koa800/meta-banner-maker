# アドネス株式会社 請求書提出スキル

## 概要

毎月の業務委託報酬と経費立替の請求書を作成し、アドネス株式会社に提出する。

## タイミング

- **毎月初（翌月5日まで）** に前月分を提出
- 例: 1月分 → 2月5日までに提出

## 手順

### 1. 単価の確認

請求金額管理シートを開き、**甲原海人の行（行5）** の該当月列を確認する。

- 列の見方: ヘッダー行4の「YY/MM 末払い」が支払月。**前月分の報酬** がその列に記載
  - 例: 「26/2 末払い」= 1月分報酬
- この金額を業務委託報酬の単価として使用

### 2. INVOYで請求書作成（2通）

1. INVOY にログイン（Google: koa800sea.nifs@gmail.com）
2. 「発行」→「請求書」から **アドネス株式会社** のテンプレートを **複製**
   - 業務委託報酬と経費建替の2種類がある。それぞれ複製する

#### 業務委託報酬

| 項目 | 設定値 |
|------|--------|
| 発行日 | 作成日（当日） |
| お支払い期限 | 作成日の月の最終日 |
| 件名 | `業務委託報酬　YYYY年MM月分` |
| 品目明細（単価） | スプレッドシートの金額 |

#### 経費立替

| 項目 | 設定値 |
|------|--------|
| 発行日 | 作成日（当日） |
| お支払い期限 | 作成日の月の最終日 |
| 件名 | `経費立替　YYYY年MM月分` |
| 品目明細（単価） | ¥2,000（Facebook広告代、毎月固定） |

### 3. PDF保存

各請求書をPDFで保存する。

### 4. 甲原さんに確認（必須）

**提出前に必ず承認を得る。**

- LINE で2つのPDFを送信して確認してもらう
- **承認されるまで提出しない**

### 5. 提出フォームで提出（2回）

承認後、以下のフォームで **業務委託報酬** と **経費立替** をそれぞれ1回ずつ提出する。

| フォーム項目 | 入力値 |
|-------------|--------|
| メールアドレス | koa800sea.nifs@gmail.com |
| アドネスの誰が担当か？ | 三上功太 |
| 請求書発行事業者の氏名 | 甲原海人 |
| 提出者の氏名 | 甲原海人 |
| 請求金額 | PDFに記載の金額（業務委託と経費で異なる） |
| 備考 | 未記入 |
| PDFファイル追加 | 対応するPDFを添付（業務委託と経費を間違えない） |

### 6. Google Drive に保存

保存先: `マイドライブ > アドネス > 給与、経費 > [YYYY]年/`
- フォルダID: `1bgUmZAH6okAGUWQ7uYSUo4K2KbOomfaM`

#### ファイル名の命名規則

| 種類 | ファイル名 |
|------|-----------|
| 業務委託報酬 | `YYYY年M月請求書_甲原海人.pdf` |
| 経費立替 | `YYYY年M月経費建替_甲原海人.pdf` |

例: `2026年1月請求書_甲原海人.pdf`, `2026年1月経費建替_甲原海人.pdf`

### 7. 完了報告

甲原さんに提出完了を報告する。

## 注意事項

- **承認前に絶対に提出しない**（ステップ4の確認が必須）
- フォーム提出時、業務委託と経費のPDFを取り違えないこと
- 経費立替は基本的に毎月 ¥2,000（Facebook広告代）で固定
