すべての制作物（バナー広告・動画広告・LP・資料・SNS投稿・メール等）に共通するデザインの基礎原則と判断基準。

## 1. デザインの4原則

デザインの良し悪しは感覚ではなく、4つの原則で説明できる。すべての制作物でこの4つを意識的に適用する。

### 1-1. 近接（Proximity）

**関連する要素を近づけ、関連しない要素を離す。**

- 情報のグループが視覚的に伝わるかどうかは「距離」で決まる
- 商品画像・商品名・キャッチコピーは密集させる → ユーザーが瞬時に「これは1セットの情報だ」と認識できる
- 関連のない情報との間には十分な余白を設ける
- グループの数が多すぎると逆効果。情報を整理してからレイアウトする

**判断基準**: 初見の人が「どれとどれがセットか」を0.5秒で判断できるか？

### 1-2. 整列（Alignment）

**すべての要素を意図的に揃える。何も「なんとなく」置かない。**

- テキスト・画像・ボタンなど、見えない線（グリッド）に沿って配置する
- 左揃え・中央揃え・右揃えを混在させない。1つのブロック内では揃え方を統一する
- 要素同士の繋がりが直感的に伝わり、全体に統一感が生まれる
- 少しでもズレていると「雑」「素人っぽい」という印象を与える

**判断基準**: 要素の端を線で結んだとき、きれいに1本の線になるか？

### 1-3. 反復（Repetition）

**同じルールを繰り返し使い、一貫性を作る。**

- 色・フォント・アイコン・余白のサイズ・装飾スタイルを統一ルールで繰り返す
- 反復によって得られる3つの効果:
  - **現在地認識**: 「今どこを見ているか」が分かる（サイト内のナビゲーション等）
  - **強調効果**: パターンの中で1つだけ違う要素があると、それが際立つ
  - **緩急**: 繰り返しの中にリズムが生まれ、単調さを避けられる
- 「このデザイン、統一感がある」と感じるのは反復が効いている証拠

**判断基準**: 同じ役割の要素が、すべて同じ見た目になっているか？

### 1-4. 対比（Contrast）

**重要な情報とそうでない情報に、明確な強弱をつける。**

- サイズ・色・太さ・余白で視覚的な優先順位を伝える
- 近接・整列・反復で情報を整理すると、全体が均一になりやすい。対比で意図的にメリハリをつける
- 「ちょっとだけ違う」は逆効果。やるなら大胆に差をつける
- セール広告の「●%OFF」を大きく表示するのが典型例

**判断基準**: 最も伝えたい情報が、最も目立っているか？

## 2. 4原則の適用順序

4原則は独立しているが、実務では以下の順序で適用すると崩れにくい。

```
① 近接: 情報をグループ化する
② 整列: グループ内・グループ間を揃える
③ 反復: 色・フォント・装飾のルールを統一する
④ 対比: 優先度の高い情報を強調する
```

①〜③で整理した後に④で強弱をつける。順序を逆にすると「派手だけど読みにくい」デザインになる。

## 3. デザイン基礎用語

制作物のレビューや指示で使う基本用語。正しく理解していないとコミュニケーションがズレる。

> **4原則との違い**: 4原則（近接・整列・反復・対比）はすべての制作物で必ず適用する絶対ルール。一方、以下の用語に含まれる数値や比率（配色の70:25:5等）はあくまで目安であり、制作物のコンセプトや目的によって柔軟に判断する。原則を知った上で、意図を持って崩すのはOK。知らずに外れるのはNG。

### 3-1. 余白（ホワイトスペース）

- 要素と要素の間の「何もない空間」
- 余白の役割:
  - **整理**: 情報のグループを分ける（近接の原則と連動）
  - **視線誘導**: 余白の方向に目が流れる
  - **強調**: 余白で囲まれた要素は目立つ
- 紙面やスクリーンを情報で埋め尽くすと、すべてが埋もれて何も伝わらない
- 余白は「何もない」のではなく「意図的に空けている」

### 3-2. カーニング

- 文字と文字の間隔を微調整する作業
- フォントの特性により、文字の組み合わせによって間隔が不均一に見えることがある
- 特に大きな見出しやロゴでは、カーニングの雑さが目立つ
- 本文テキストでは自動カーニングで十分。見出し・タイトルは手動調整する

### 3-3. フォントと書体

- **書体**: 文字スタイルの分類体系（明朝体、ゴシック体、セリフ体、サンセリフ体など）
- **フォント**: 同一デザイン・同一サイズの文字セット。書体の具体的な製品
- 制作物で使うフォントは2〜3種類に絞る（反復の原則）。増やすほど統一感が崩れる

### 3-4. 配色

- 色は心理的な影響を与える。選択には根拠が必要
- 基本の3要素: **色相**（何色か）・**明度**（明るさ）・**彩度**（鮮やかさ）
- 配色の比率ルール: **ベースカラー 70% : メインカラー 25% : アクセントカラー 5%**
- アクセントカラーがCTAボタンや強調箇所に使われることが多い（対比の原則と連動）
- 色数を増やすほどデザインは難しくなる。迷ったら色数を減らす

## 4. 制作物別の適用ポイント

4原則は共通だが、制作物ごとに特に重視すべきポイントが異なる。

| 制作物 | 特に重要な原則 | ポイント |
|--------|--------------|---------|
| バナー広告 | 対比 > 近接 | 一瞬で伝わる必要がある。情報量を絞り、最も伝えたい1メッセージを大胆に強調する |
| 動画広告 | 反復 > 対比 | テロップ・色使い・トランジションのルールを統一。変化する瞬間に対比を使う |
| LP | 整列 > 近接 | 縦長のページで情報が多い。グリッドで揃え、セクションごとにグループ化する |
| 資料・スライド | 整列 > 反復 | ページ間の統一感が重要。テンプレートで反復を担保する |
| SNS投稿 | 対比 > 反復 | フィード上で目を止めてもらう必要がある。アカウント全体で反復、個々の投稿で対比 |
| メール | 近接 > 整列 | 読み飛ばされやすい。情報をグループ化し、余白で区切る |

## 5. デザインレビューチェックリスト

制作物を確認するとき、以下の順でチェックする。

### 4原則チェック

- [ ] **近接**: 関連情報がグループ化されているか。無関係な要素が近すぎないか
- [ ] **整列**: すべての要素が見えない線に沿って揃っているか。ズレがないか
- [ ] **反復**: 色・フォント・装飾が統一ルールで使われているか。例外がないか
- [ ] **対比**: 最も伝えたい情報が最も目立っているか。強弱が中途半端でないか

### 基礎チェック

- [ ] **余白**: 情報が詰め込まれすぎていないか。余白が意図的に使われているか
- [ ] **フォント数**: 3種類以内に収まっているか
- [ ] **配色**: 色数が多すぎないか。70:25:5の比率から大きく外れていないか
- [ ] **視線の流れ**: 目が自然に重要な情報へ導かれるか

### 目的チェック

- [ ] **誰に向けたものか**が明確か（ターゲット）
- [ ] **何を伝えたいか**が1つに絞られているか（メッセージ）
- [ ] **どう行動してほしいか**が明確か（CTA）
- [ ] 初見の人が3秒で「何のことか」分かるか

## 6. よくあるNG（アンチパターン）と改善方法

| NGパターン | 原因（どの原則に反しているか） | 改善方法 |
|-----------|--------------------------|---------|
| 情報がバラバラに散らばっていて何がセットか分からない | 近接の不足 | 関連情報を物理的に近づけ、グループ間の余白を広げる |
| テキストや画像の端が揃っていない | 整列の不足 | グリッドを設定し、すべての要素を揃える |
| ページごとに色やフォントが違う | 反復の不足 | スタイルガイドを作り、全ページで統一する |
| 全体が均一で、どこを見ればいいか分からない | 対比の不足 | 最重要情報のサイズ・色・太さを大胆に変える |
| 余白がなく窮屈 | 余白の軽視 | 要素を減らすか、余白を意図的に確保する |
| 色が多すぎてゴチャゴチャ | 配色ルール違反 | 3色に絞り、70:25:5で配分する |
| 「ちょっとだけ」サイズや色が違う要素がある | 対比の中途半端 | 同じにするか、大胆に差をつけるか、どちらかに振り切る |
