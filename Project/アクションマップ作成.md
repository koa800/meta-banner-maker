## AM命名ルール

- **動詞**: 「つくろう！」で統一（「作成しよう」「作ろう」「構築しよう」は使わない）
- **語尾**: 「しよう！」「書こう！」「磨こう！」等は文脈に応じてOK
- **生成AI系コース**: 「生成AIで〜」で統一（「AIで〜」は使わない）
- **文字数**: 18文字以内を目安。超える場合は意味を損なわない範囲で簡潔に
- **具体性**: タイトルだけで何をするか分かること（「社内ツール」「仕組み化」等の曖昧表現はNG）
- **コース表示名**: 「導線構築コース」→「導線構築代行コース」として表示

## 技術メモ

### 反映スクリプト（reflect_to_suuchi.py）

- 再構成シートのAM名を数値シートに反映
- 既存の運用データ（D〜L列）はAM名マッチングで引き継ぎ
- `RENAME_MAP`: ツール名削除によるAM名変更のマッチング用
- `COURSE_MATCH`: コース名変更のマッチング用（エンジニアコース↔プログラミングコース等）

### Google Sheets API操作パターン

```python
# セル値の一括更新
from gspread import Cell
cells = [Cell(row=i, col=2, value='新しいAM名') for i in rows]
ws.update_cells(cells, value_input_option='RAW')

# テキスト色の変更（batch_update）
sh.batch_update({'requests': [{
    'repeatCell': {
        'range': {'sheetId': 484745247, 'startRowIndex': row-1, 'endRowIndex': row,
                  'startColumnIndex': 1, 'endColumnIndex': 2},
        'cell': {'userEnteredFormat': {'textFormat': {'foregroundColor': {'red': 1.0, 'green': 0, 'blue': 0}}}},
        'fields': 'userEnteredFormat.textFormat.foregroundColor'
    }
}]})

# 行の挿入
sh.batch_update({'requests': [{
    'insertDimension': {
        'range': {'sheetId': 484745247, 'dimension': 'ROWS',
                  'startIndex': row_0based, 'endIndex': row_0based + 1},
        'inheritFromBefore': True
    }
}]})

# セル結合の解除・再設定
sh.batch_update({'requests': [
    {'unmergeCells': {'range': {...}}},
    {'mergeCells': {'range': {...}, 'mergeType': 'MERGE_ALL'}}
]})
```

### 注意事項

- A列はセル結合されている。コース分割等でA列を書き換える場合は `unmergeCells` → `mergeCells` の手順が必要
- `ws.batch_update()` と `sh.batch_update()` は異なるAPI。セル値更新は `ws.update_cells()`、フォーマット変更は `sh.batch_update()` を使う
- 行を挿入/削除すると以降の行番号がずれるため、下から操作するか一括で処理する
