# LP自動生成システム — 開発・デザイン依頼ブリーフ

## これは何か

AIがメールマーケティングを自動で行うシステムの一部として、**ランディングページ（LP）を自動生成する仕組み**を作りたい。

---

## 背景（なぜ必要か）

アドネスでは現在、以下の流れでマーケティングを行っている:

```
広告 → LP → メール取得 → メール配信 → LINE登録 → セミナー → 購入
```

このうち「メール配信」と「LP作成」を**AIで自動化**したい。

メール配信は Mailchimp API で自動化できることが確認済み。
問題は **LP作成**。現在使っている UTAGE や Studio にはページを外部から自動生成するAPIがないため、**自前でLPを生成・公開する仕組み**が必要。

---

## ゴール

**「AIに要件（誰に・何を・どんなアクションを取ってほしいか）を伝えたら、それなりにクオリティの高いLPが自動で生成・公開される」** という状態を作る。

---

## 具体的に作ってほしいもの

### 1. LPテンプレート（デザイナーへ）

以下の用途別に、**HTMLテンプレートを数パターン**作成してほしい。

| テンプレート種別 | 用途 | 主な構成要素 |
|----------------|------|------------|
| セミナー告知LP | セミナー・ウェビナーへの参加申込 | ヘッドコピー / 日時・内容 / 講師紹介 / 申込ボタン |
| 商品紹介LP | 商品・サービスの紹介→購入 | ヘッドコピー / 悩み提起 / 解決策 / 特徴 / 価格 / CTAボタン |
| オプトインLP | メールアドレス or LINE登録の獲得 | ヘッドコピー / ベネフィット / 登録フォーム or LINE登録ボタン |
| コンテンツ配信LP | 動画・記事・PDF等の提供 | ヘッドコピー / コンテンツ埋め込み / 次のアクションへのCTA |

**デザイン要件:**
- アドネスのブランドに合ったデザイン（既存のUTAGE LPのトーンを参考に）
- スマホファースト（閲覧の大半がスマホ）
- **差し替え可能な箇所を明確に**（後述の「変数」で、AIがテキスト・画像・URL等を差し込む）

### 2. テンプレートエンジン（エンジニアへ）

上記テンプレートに対して、**変数を渡すとHTMLが生成される仕組み**を作ってほしい。

```
入力（AIが生成する内容）:
  - headline: "たった3STEPで月収50万円を達成する方法"
  - sub_headline: "年商30億の社長が教える再現性のあるロードマップ"
  - body_sections: [悩み提起, 解決策, 実績, ...]
  - cta_text: "無料セミナーに申し込む"
  - cta_url: "https://lin.ee/xxxxx"（LINE登録URL）
  - image_urls: [hero画像, 講師画像, ...]
  - template_type: "seminar"

出力:
  - 完成したHTML（1ファイル）
  - 公開URL（例: https://lp.addness.co.jp/ai-seminar-20260315）
```

**技術要件:**
- テンプレートはHTML + CSS（フレームワーク不問。Jinja2, Handlebars等なんでもOK）
- 生成されたHTMLを自動でサーバーにデプロイ → URLが発行される
- ホスティングは Render（既に利用中）を想定。他でもOK
- OGP（SNSシェア時の画像・タイトル）も変数で差し込めるように
- Google Analytics / Meta Pixel のトラッキングコードを埋め込めるように

### 3. 全体フロー（エンジニアへ）

最終的にこういう流れで動く:

```
① AIが要件を受け取る（セグメント情報 + マーケティング目的）
    ↓
② AIがメール本文とLP内容を同時にライティング
    ↓
③ テンプレートエンジンがLPのHTMLを生成
    ↓
④ サーバーにデプロイ → URL発行
    ↓
⑤ メール本文にLP URLを埋め込み → Mailchimp APIでキャンペーン作成
    ↓
⑥ 承認後 → 配信
```

③④の部分を作ってほしい。①②⑤⑥は別途こちらで実装する。

## 参考: 現在のLPの例

UTAGEで作成している既存LPのデザインを踏襲したい。以下のパターンが多い:

- 縦長1カラム
- ファーストビュー: 大きなヘッドコピー + サブコピー + CTA
- スマホ最適化（横幅100%）
- セクション区切り: 背景色の切り替え
- CTAボタン: 目立つ色（緑 or オレンジ系）、複数箇所に配置
- フォント: ゴシック体ベース
