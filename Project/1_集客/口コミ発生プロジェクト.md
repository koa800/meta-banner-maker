# 口コミ発生プロジェクト

最終更新: 2026-03-03

## 概要

スキルプラスのオンボーディング完了直後のユーザーに、Xで口コミ投稿を促す仕組み。
リンクをクリックするだけでXの投稿画面に飛び、ランダムな文章が入った状態でそのまま投稿できる。

## 導線

1. ユーザーがオンボーディングフローを完了（エネルギーが高い状態）
2. 「今の気持ちを言葉にしよう」と訴求
3. SNS投稿した人にスタバチケットをオファー
4. リンクをクリック → Xの投稿画面に直接遷移（ランダム文章入り）
5. そのまま投稿 or 編集して投稿

## URL

- **公開URL**: https://koa800.github.io/skillplus-review/
- **GitHubリポジトリ**: https://github.com/koa800/skillplus-review
- **L-Stepに設置**: 上記URLをL-Step内のオンボーディングフローに配置

## 技術構成

- **ホスティング**: GitHub Pages（無料・サーバーレス・メンテナンス不要）
- **構成**: 静的HTML 1ファイルのみ（`index.html`）
- **仕組み**: JavaScriptでランダム文章を選択 → `x.com/intent/post` にリダイレクト
- **フォールバック**: `twitter.com/intent/tweet`（旧ドメイン）/ `<noscript>` 対応 / 3秒タイムアウト
- **キャッシュ対策**: `Cache-Control: no-cache` メタタグ

## 文言の方針

- ハッシュタグなし（Xの検索に引っかからなくなるため）
- 絵文字は✨のみ、半数程度に控えめに
- ステマっぽくない自然な一般ユーザーのトーン
- オンボーディング直後の「やるぞ！」という決意表明

## 文章の追加・変更方法

1. `System/skillplus_review/index.html` の `messages` 配列を編集
2. `/tmp/skillplus-review` にコピーして `git push`（またはGitHub上で直接編集）
3. GitHub Pagesが自動で再ビルド（1〜2分で反映）

## 運用上の注意

- サーバー・DB・APIキー不要。完全に静的なので止まるリスクは極めて低い
- GitHub Pagesの稼働率は99.9%+
- ソースは2箇所に保持（メインリポジトリ + GitHub Pages用リポジトリ）
