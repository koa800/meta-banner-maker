# 定期チェック項目

> **目的**: 定期的に振り返り、改善するための「心拍」を定義する
> **読み込み対象**: メインエージェントのみ
> **更新頻度**: 中（チェック項目を見直したら更新）

---

## 📋 このファイルについて

このファイルは、定期的な**振り返り**と**改善**のためのチェック項目を定義します。

- **日次チェック**: 毎日確認すること
- **週次チェック**: 毎週確認すること
- **月次チェック**: 毎月確認すること

このファイルを読み込んだエージェントは、定期的にあなたに「振り返り」を促します。

---

## 🔄 フィードバックループの実行

### 計画 → 実行 → 検証 → 改善

#### 今月のPDCAサイクル

```
【計画】
目的: [何を達成したいか]
仮説: [こうすれば、こうなるはず]
使った判断軸/パターン: [どの判断軸やパターンを使ったか]

【実行】
実際にやったこと: [具体的な行動]
実行時の感情: [どんな感情状態だったか]

【検証】
結果: [何が起きたか、数値データがあれば記載]
仮説との差異: [仮説通りだったか、違ったか]
なぜこの結果になったか: [原因の考察]

【改善】
次に活かすこと: [次回、どう改善するか]
マスターファイルの更新内容: [どのファイルを、どう更新するか]
```
