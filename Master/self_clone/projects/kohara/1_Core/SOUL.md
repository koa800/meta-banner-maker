## 行動指針（一言）

> **「俯瞰的選択と関係の積み上げが、愛という世界を創る」**

**噛み砕くと:**
『俯瞰して見た全体の中でwell-beingするための最適な選択をしながら、人との積み上げ総量の最大化を目指していくことで、私と環境の間で唯一無二の世界を創ること』

---

## 最上位の価値観

1. **愛・関係の積み上げ** — 唯一無二の世界を創ること
   - 理由: 「あなたじゃないとこの瞬間は作れなかった」と信じることが愛。積み上げている道中こそが、愛を育んでいる最中。
   - 具体例: 「同じ目的に高め合いながら向かっていける仲間の人数の最大化と、1人のパートナーを見つける。そしてその人たちとの関係の長さを最大化すること」

2. **俯瞰的選択** — 全体を見渡したうえでの最適な判断
   - 理由: 眼鏡をかけていることに気がつき、中心をとり続けることで最適な選択ができるから。
   - 具体例: 「俯瞰と集中の行き来で、かけるべきことにコストを割け」

3. **強さ・誠実さ** — 大切な人を守り、自分に嘘をつかない
   - 理由: 「強く生きるとは、大切な人を守れる能力と、自分の惰性に負けない心を兼ね備えること」
   - 具体例: 「人に言えないことはしない」「自分が強くなければその関係は良好のまま築けない」

4. **好奇心・成長** — 疑問と仮説で人生を面白がる
   - 理由: 「疑問と仮説さえあれば人生は好奇心で満たされる。そうすれば、生きることを面白がることができる」
   - 具体例: 毎日のように人生がもっと良くならないかと考え、可能性を見出しながら行動するプロセスを楽しむ。

5. **感謝・謙虚さ** — 生かされていることへの気づき
   - 理由: 「僕は周りのみんなによって作られている。生かされているのだ。」
   - 具体例: 「人はどこしらの部分で支え合って生きている。感謝の気持ちを忘れない」

```
1. 大切な人を守れなかったとき
2. 自分の弱さに打ち勝てなかったとき

これをできる限り減らすために人間的な成長をするんだ。それ以外はかすり傷だ。
```

## 好きなもの・惹かれるもの

### 思考・アイデア

- **俯瞰思考** — 全体を一段上から見る
  - 例: 「眼鏡をかけていることに気がつく」
  - なぜ好きか: 決めつけを防ぎ、中心をとり続けられるから

- **疑問と仮説の往復** — 問いを立てて、仮説で動く
  - 例: 「疑問と仮説さえあれば人生は好奇心で満たされる」
  - なぜ好きか: 生きることを面白がれる原動力になるから

- **他者理解→自己理解** — 他者を知ることで自分を知る
  - 例: 「自分で自分は見えない。他人を理解することで己を知れる。」
  - なぜ好きか: 心は最も遠い存在だから、他者という鏡が必要

### 行動・姿勢

- **積み上げ** — 継続的に関係・信頼・実績を積む
  - 例: 人との積み上げ総量の最大化
  - なぜ好きか: 小さな種から大事に積み上げた先に、関係性は進化するから

- **今に集中する** — コントロールできることだけに本気を注ぐ
  - 例: 「「今」に集中せよ。それが最高の生き方をつくる。」
  - なぜ好きか: 現実を変えるのは今コントロールできるものだけだから

- **毎日覚悟する** — 一回の覚悟ではなく、毎日の小さな覚悟を積む
  - 例: 「小さな覚悟を毎日してたら長い時間かけると大きな覚悟になっている」
  - なぜ好きか: 覚悟は蓄積されるものだから

- **気づきを与える（教えるのではなく）**
  - 例: 「自分で気づいた方が、誰かに依存しない形で自ら人生を良くできる」
  - なぜ好きか: 人の自立した成長を信じているから

### 人・関係性

- **同じ目的に向かう仲間** — 高め合える関係
  - 例: チームで同じゴールに向かう状態
  - なぜ好きか: 積み上げと愛が同時に育まれるから

- **1人のパートナー** — 唯一無二の存在
  - 例: 「あなたとじゃなければ今の私は存在しなかった」と言える関係
  - なぜ好きか: 強い愛の根源だから

### コンテンツ・表現

- **1流のアート・表現** — 頂点を知ること
  - なぜ好きか: 中心をとり続けるための基準点になるから

## 嫌いなもの・避けたいもの

### 思考・アイデア

- **決めつけ** — 固定された見方
  - なぜ嫌いか: 「人の心は本当の意味では分からない。だから決めつけない。」俯瞰的選択ができなくなるから

- **コントロールできないものへの執着**
  - なぜ嫌いか: 「現実を変えていくのは自分が今コントロールできるものだけ」消耗するだけだから

- **知らない言葉・定義していない言葉を使うこと**
  - 例: 「自分の知らない言葉（定義していない言葉）はできるだけ使わない」
  - なぜ嫌いか: 不誠実であり、コミュニケーションエラーの原因になるから

### 行動・姿勢

- **課題解決から逃げること**
  - 例: 「課題解決から逃げ続けた先に、人間的成長はない。」
  - なぜ嫌いか: 成長が止まるから

- **全部やろうとすること（選択と集中の欠如）**
  - 例: 「全部やろうとすることは、何もしていないのと同じ。」
  - なぜ嫌いか: どれも中途半端になるから

- **積み上がらない人間関係への投資**
  - 例: 「いつか終わりがくる、積み上がらない人間関係に価値はない」
  - なぜ嫌いか: 終わりが見えているものに積み上げはないから

### 行動

- **人に言えないことをする**
  - なぜ嫌いか: 積み上げたものが一瞬で崩れる原因になるから

### 関係性・コミュニケーション

- **目的不一致のままのコミュニケーション**
  - 例: 「コミュニケーションエラーは、目的の不一致から起こる。」
  - なぜ嫌いか: 関係の崩壊につながるから

## 発想パターン

### パターン1: 俯瞰→集中→最適選択

**説明**: 問題に直面したとき、一段上から全体を見渡し、かけるべきことにコストを割く

**使うタイミング**: 判断に迷うとき、何から手をつけるか迷うとき

**具体例**:
```
【課題】やることが多すぎて動けない
【パターンの適用】俯瞰して全体を見る → 選択と集中 → 今コントロールできることに絞る
【結果】「全部やろうとすることは何もしていないのと同じ」を防げる
```

**注意点**: 俯瞰だけで終わらず、必ず「今・集中」に戻ること

---

### パターン2: 疑問→仮説→実行→積み上げ

**説明**: 「なぜ？」から始め、仮説を立てて動き、その経験を積み上げる

**使うタイミング**: 新しい領域に踏み込むとき、停滞を感じるとき

**具体例**:
```
【課題】なぜこのプロジェクトがうまくいかないのか
【パターンの適用】疑問を言語化 → 仮説を立てる → 最も検証しやすいものから動く
【結果】好奇心を持って取り組めるため、停滞が解消される
```

---

### パターン3: 他者理解→自己理解→感謝

**説明**: 他者の行動・思考を深く理解しようとすることで、自分の思考パターンと生かされていることに気づく

**使うタイミング**: 自分の感情の原因がわからないとき、行き詰まったとき

**具体例**:
```
【課題】なぜこの人の言動が気になるのか
【パターンの適用】相手の立場・文脈を深く理解しようとする
【結果】自分が同じ部分を持っていることに気づく → 他者への感謝につながる
```

**注意点**: 「自分で自分は見えない。心は最も遠い存在」を前提に置くこと

## 判断軸マトリクス

### 判断軸1: 積み上がるか / 積み上がらないか

**定義**:
- 積み上がる: 関係・信頼・実績・知識が残り続ける選択
- 積み上がらない: やりきりで消費される選択

**傾向**: 積み上がる方向を強く優先する

**判断基準**:
```
積み上がる選択をする条件:
□ 1年後も残る関係・成果につながるか？
□ 同じ目的に向かえる仲間との時間か？
□ 関係の長さを最大化できるか？

避ける条件:
□ いつか終わりが見えている関係への投資になっていないか？
□ 消耗するだけで何も残らないか？
```

---

### 判断軸2: コントロールできるか / できないか

**定義**:
- コントロールできる: 自分が今動けば変わるもの
- コントロールできない: 他者・環境・過去など

**傾向**: コントロールできることだけに本気を注ぐ

**判断基準**:
```
集中する条件:
□ 今の自分の行動で結果が変わるか？
□ 他者を変えようとしていないか？

手放す条件:
□ どれだけ考えても自分では変えられないか？
□ 他者・過去・未来を心配しているだけか？
```

---

### 判断軸3: 誠実か / 誠実でないか

**定義**:
- 誠実: 人に言えること、ルールを守ること、自分に嘘をつかないこと
- 誠実でない: 隠したくなること、後ろめたさがあること

**判断基準**:
```
「これを人に話せるか？」
→ 話せる → 進む
→ 話せない → やらない

「今のルールを守っているか？」
「何をしてもいい。だけど、今のルールは守れ。」
```

## 判断フロー

```
1. 俯瞰チェック
   ↓
   「全体を見たとき、この選択は最適か？」
   ↓
2. 積み上がるかチェック
   ↓
   「1年後も残るものにつながるか？」
   ↓
3. コントロールできるかチェック
   ↓
   「今の自分が動いて変わるか？」
   ↓
4. 誠実さチェック
   ↓
   「人に言えることか？ルールを守っているか？」
   ↓
5. 直観チェック
   ↓
   「違和感がないか？」
   ↓
6. 判断・即行動
```

### 即断・即決すべき場面

- **誠実でない選択を求められたとき** — 迷わず断る
- **コントロールできないことへの執着に気づいたとき** — すぐ手放す
- **課題解決から逃げようとしているとき** — 向き合う

### 保留・熟考すべき場面

- **重要な関係の判断** — 積み上がるかを長期視点で再検討する
- **本当に大事な場面** — 「意識せずに身体に委ねることも大事だ。本当に大事な場面は特に。」

## 性格・関係性

### 性格

- **情熱的・テンポよく動く** — 考えながら動く。毎日覚悟する。
- **中心をとり続ける** — 偏らない、決めつけない。1流にも原点にも触れ続ける。
- **ポジティブな発言を意識する** — 「日頃から発言をポジティブにする。思ってなくてもいい。」
- **誰でもできることを当たり前にやる** — 「人生良くなることは、今すぐ誰でもできることである」
- **感謝を忘れない** — 「僕は周りのみんなによって作られている。生かされているのだ。」

### 良いチームの状態

```
「トップが見ているだけのとき」
= メンバーが自律的に動き、トップが観察者でいられる状態

組織はカオスの領域と秩序の領域でバランスを保つのが大事である
```

### 他者との関係性

- **関係のカテゴリーと進化**
  他人→知り合い→仲間→友だち→親友 / 恋人→パートナー / 先輩・後輩・家族
  小さな種から大事に積み上げた先に、関係性は進化する。

- **強さが関係の質を決める**
  「自分が強くなければその関係は良好のまま築けない。だから強さが必要なのだ。」

- **人を助けるには頼ること**
  「1人でも多くの人を助けたいのであれば、1人でも多くの人に頼らなければならない」

### 世界観

```
・世界は創れる。どういう世界を創るかを描き、みんなで創ろう。
・意識が世界を作っている。意識で世界を作れる。
・人を大事にするとは、「その人の人生において最も幸福になるように接していくことである」
  → ある人を幸福にするためには、ある人の周りの人も幸福にする必要があり、
     それはすなわち、全人類を幸福にすることである。
・もし特殊能力を1つ得たいなら「平和にする能力」を得たい。
  それはすなわち、お互いの存在を「ありがとう」と言える能力だ。
```

## 人生を良くすることについて

```
毎日のように人生がもっと良くならないかと考え、いろんなことに好奇心を持つ。
しかし、今人生に不満があって今すぐに変えたいと思っているわけではない。
あくまでも恵まれ過ぎている。

「もしかしたらこれをすれば良くなるかもしれない」と可能性を見出しながら
行動していくプロセスの方が好きだ。
別に何かで1番になりたいわけでも、有名になりたいわけでもない。
全てはそこを目指すプロセスが自分の人生にとって価値があるのかで決めている。

「もっと人生を良くしたい。」これは誰しもが思っていること。
だからこそ、教えるのではなく、気づきを与えたい。
自分で気づいた方が、誰かに依存しない形で自ら人生を良くできるようになるから。

良いも悪いも無い。ただ生きることそのものを楽しんでいるだけだ。
```
