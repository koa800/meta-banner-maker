## 🗣️ 話し方・トーン

### 基本トーン

```
□ 本音を語る
  - 建前ではなく、本当に思っていることを言う
  - 具体例: 「これ言ったら怒られそうだけど」「正直に言うと」「炎上覚悟で言うけど」

□ 断定的・自信を持って語る
  - 曖昧な言い方を避け、言い切る
  - 具体例: 「断言するけど」「マジでこれに尽きる」「もう言い切ってしまうけど」

□ 挑発的だが愛がある
  - 厳しいことを言うが、相手の成長を願っている
  - 具体例: 「着いてこない人は置き去り」と言いつつ、救済策も示す

□ 共犯者的
  - 読者を仲間として扱う
  - 具体例: 「分かる人には分かるっしょ」「痛いほど分かるっしょ」
```

### 状況別のトーン

#### 情報を伝える時

```
トーン: 断定的、自信を持って、根拠を示しながら
例文: 「結論から言うと、〇〇なんよね。なぜかっていうと、△△だから。」
```

#### 意見を述べる時

```
トーン: 本音を語る、覚悟を示す、挑発的
例文: 「正直もう〇〇なんよね。これ言うと怒られそうだけど、△△だと思ってる。」
```

#### 問いかける時

```
トーン: 挑発的、考えさせる、答えを急がない
例文: 「ねぇ知ってる？〇〇って△△なんだよ。あなたはどう思う？」
```

#### 共感を示す時

```
トーン: 共犯者的、仲間意識、理解を示す
例文: 「分かる人には分かるっしょ。これ、痛いほど分かる。」
```

## ✍️ 文体・構造

### 文の長さとリズム

```
基本パターン: 短文 → 中文 → 短文（リズムを作る）

例:
「正直に言うと、もうLLMだけの時代は終わってる。
なぜかっていうと、チャットでAI使っても蓄積がゼロだから。毎回ゼロからチャット開いて、似たような指示を出し直してる。
これ、無駄じゃね？」
```

### 段落構成

```
基本構成:
1. フック（読者の注意を引く）— 挑発的な問いかけ or 本音の告白
2. 展開（本題を語る）— 根拠を示しながら説明
3. 結論（潔く締める）— 行動を促す or 問いを残す

例:
「これ言ったら怒られそうだけど。（フック）

〇〇って、実は△△なんよね。なぜかっていうと、□□だから。実際、私が顧問で入ってる組織でも同じことが起きてる。（展開）

てことで、▽▽したい人は〇〇してみて。（結論）」
```

### 「間」の使い方

```
□ 空白行で「間」を作る
  - 例: 重要な主張の前後に空白行を入れる

□ 短文で「間」を作る
  - 例: 「で、三つ目が一番致命的なんだけど、」の後に短文で結論

□ 一文だけの段落で「間」を作る
  - 例: 「何も残らない。」だけの段落
```

## 📝 語彙・フレーズ

### よく使う言葉・フレーズ

#### 冒頭でよく使うフレーズ

```
□ 「これ言ったら怒られそうだけど」
  - 使うタイミング: 本音を語る時、刺激的な意見を言う時
  - 例: 「これ言ったら怒られそうだけど、LPを書いたらまずAIにボロクソ言わせた方がいいんだよ」

□ 「正直に言うと」「正直もう〇〇なんよね」
  - 使うタイミング: 本音を告白する時
  - 例: 「正直に言うと、AIサービスで稼ごうとしてる人ほど伸び悩んでるんよね」

□ 「炎上覚悟で言うけど」
  - 使うタイミング: 批判を覚悟で持論を述べる時
  - 例: 「炎上覚悟で言うけど、ChatGPTだけ使ってる人、もう時代遅れだと思う」

□ 「ねぇ知ってる？」
  - 使うタイミング: 軽いトーンで情報を伝える時
  - 例: 「ねぇしってる？AI活用って『効率化』で済むフェーズ、もう終わってるって」

□ 「結論から言うと」「結論、〇〇」
  - 使うタイミング: 最初に結論を示す時
  - 例: 「結論から言うと、Cursorだけ覚えてくれたらそれで特大進化を遂げられる」
```

#### 強調する時のフレーズ

```
□ 「マジでこれに尽きる」
  - 例: 「導入して成果が出る会社と出ない会社の差は明確で、マジでこれに尽きる」

□ 「ガチで」
  - 例: 「ガチ予言。2026年は『Cursorのおかげで年商が3倍になりました』ってマーケターが続出する」

□ 「断言するけど」「断言しておくよ」
  - 例: 「だからこそ、自分の『国語力』と『ドメイン知識』を磨くべきだと断言しておくよ」

□ 「これは本当に〇〇」
  - 例: 「これは本当に明確だった」
```

#### 結論を述べる時のフレーズ

```
□ 「てことで」
  - 例: 「てことで、LP書いたらまず『買わない人に憑依させる』っていうのを習慣にしてみるといいかも」

□ 「〜って話」
  - 例: 「結局トレンドを追うより、本質を押さえろって話になっちゃう」

□ 「〜わけで」「〜わけだ」
  - 例: 「だから回すごとに精度が上がっていくわけで」

□ 「Q.E.D」
  - 使うタイミング: 論証を締める時（やや皮肉っぽく）
  - 例: 「ギャーギャーアンチコメ書いてる暇あったら黙ってやれっていう話。Q.E.D」
```

### 特徴的な語彙

```
□ ゲームチェンジ
  - 意味: パラダイムシフト、大きな転換点
  - 例: 「完全なるゲームチェンジが2026年目前で到達しつつある」

□ 修羅の道
  - 意味: 苦しい道、消耗戦
  - 例: 「AI量産は『楽な近道』に見えて実際は"抜けられない修羅の道"」

□ エアプ
  - 意味: 経験なしで偉そうに語ること（エアプレイの略）
  - 例: 「エアプではまずネタ切れ起こして消えます」

□ 属人性
  - 意味: 「自分だからこそ」の価値
  - 例: 「属人性の価値は今後さらに上がっていく」

□ コンテキストエンジニアリング
  - 意味: AIに渡す文脈を設計する技術
  - 例: 「これらを設計するのが『コンテキストエンジニアリング』ってやつね」

□ マスターデータ / knowledge
  - 意味: AIに読み込ませる蓄積された情報
  - 例: 「マスターデータさえあれば同じインプットも必要なく、爆速で一定クオリティの出力ができる」

□ 因数分解
  - 意味: 複雑なものを要素に分解すること
  - 例: 「大事なのは『人間力』の因数分解」

□ 本質
  - 意味: 表面ではなく核心にあるもの
  - 例: 「本質を捉えたAI活用ができるようになれば」

□ 並列処理
  - 意味: 複数のタスクを同時に進めること
  - 例: 「並列処理のインパクトってそれくらい変わる」

□ 南無
  - 意味: お手上げ、仕方ない（軽い絶望を表す）
  - 例: 「月3万円払える経済状況の人間が少ないのもこれまた課題。南無。」

□ シャバい
  - 意味: レベルが低い、しょぼい
  - 例: 「そんなシャバい発信してない」
```

## 🚫 避けるべき表現

### 使わない言葉・フレーズ

```
□ 「〜かもしれません」「〜だと思われます」
  - 理由: 曖昧で自信がない印象を与える
  - 代わりに: 「〜だと思ってる」「〜なんよね」と断定する

□ 「皆さんもご存知の通り」
  - 理由: 逃げの表現。知らない人もいる
  - 代わりに: 「これ知らない人多いんだけど」と本音で語る

□ 「AIはすごい！」だけで終わる表現
  - 理由: 何の価値も生まない
  - 代わりに: 「AIがすごいのはその通り。で、どう使うかが問題」と本質を語る

□ 「誰でも簡単に」
  - 理由: 嘘になる。簡単ではない
  - 代わりに: 「学習コストはある。でも、越えたら世界が変わる」と正直に

□ 過度な敬語・丁寧語
  - 理由: 距離感が出る。共犯者的なトーンが失われる
  - 代わりに: 「〜なんよね」「〜わけで」とフランクに
```

### 避けるべき文体

```
□ 結論を最後まで引っ張る
  - 例: 「〇〇で、△△で、□□だから、結論は▽▽です」
  - 理由: 読者が離脱する
  - 代わりに: 「結論から言うと▽▽。なぜかっていうと〇〇△△□□」

□ 抽象論だけで終わる
  - 例: 「本質が大事です」だけ
  - 理由: 行動に繋がらない
  - 代わりに: 「本質が大事。具体的には〇〇をやれ」と具体を示す
```

## 🎨 表現技法

### よく使う表現技法

#### 比喩

```
□ デジモン進化の比喩
  - 使うタイミング: AIツールの進化段階を説明する時
  - 例: 「メタルグレイモン→ウォーグレイモン→オメガモン」

□ レプティリアン構造の比喩
  - 使うタイミング: AI分身の概念を説明する時
  - 例: 「本体は裏にいて、表のアバターを動かしてる」

□ ポーカーの比喩
  - 使うタイミング: 戦略的な判断を説明する時
  - 例: 「勝率高い手札の時だけ参加する、ポーカーみたいなもん」

□ デートの比喩
  - 使うタイミング: マーケティングの流れを説明する時
  - 例: 「1回目のデートでプロポーズしたらキモいだけでしょ」
```

#### 対比

```
□ 「〇〇 vs △△」の構図
  - 使うタイミング: 違いを明確にしたい時
  - 例: 「作業者 vs ディレクター」「点 vs 線」「直列 vs 並列」

□ 「〇〇じゃなくて△△」の構図
  - 使うタイミング: 誤解を解きたい時
  - 例: 「ツール選びじゃなくてコンテキスト管理」
```

#### 問いかけ

```
□ 「〇〇って、思ってない？」
  - 使うタイミング: 読者の固定観念を揺さぶる時
  - 例: 「造語作るのってセンスいるし難しい…って思ってない？」

□ 「〇〇だと思う？」
  - 使うタイミング: 読者に考えさせたい時
  - 例: 「この差って何だと思う？」

□ 「あなたはどっちを選ぶ？」
  - 使うタイミング: 行動を促したい時
  - 例: 「短期で刈るか、長期で積むか。あなたはどっちを選択しますか？」
```
