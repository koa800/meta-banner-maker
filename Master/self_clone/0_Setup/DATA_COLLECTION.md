# データ収集ガイド

> **目的**: セルフクローン構築に必要なデータを効率的に収集する方法を説明する

---

## 📋 なぜデータ収集が重要か？

AIクローンの精度は、**データの質と量**に比例します。

```
少ないデータ → 一般的なAI
多くのデータ → あなた独自のAIクローン
```

---

## 📚 収集するデータの種類

### 1. SNS投稿

**優先度**: ★★★★★（最重要）

**理由**:
- あなたの「素」の言葉が表れる
- 価値観、言語スタイルが明確
- 量が多く、分析しやすい

**収集方法**:

#### Twitter/X

```
方法1: アーカイブをダウンロード
1. 設定 → 「アカウント」 → 「データのアーカイブをダウンロード」
2. ダウンロードしたZIPファイルを展開
3. tweets.jsを開く
4. 過去3-6ヶ月分の投稿をコピー

方法2: 手動でコピー
1. 自分のプロフィールページを開く
2. 過去3-6ヶ月分の投稿をスクロール
3. 重要な投稿をコピー&ペースト
```

#### Facebook

```
1. 設定 → 「あなたのFacebook情報」 → 「情報をダウンロード」
2. 「投稿」を選択してダウンロード
3. 過去3-6ヶ月分の投稿をコピー
```

#### LinkedIn

```
1. 設定 → 「データのプライバシー」 → 「データのコピーを取得」
2. 「投稿」を選択してダウンロード
3. 過去3-6ヶ月分の投稿をコピー
```

---

### 2. ブログ記事・note記事

**優先度**: ★★★★☆（重要）

**理由**:
- 長文で、思考プロセスが明確
- 専門分野の知識が表れる
- 発想パターンが分析しやすい

**収集方法**:

```
1. 自分のブログ/noteを開く
2. 過去1年分の記事をリストアップ
3. 各記事の本文をコピー
4. 1つのファイルにまとめる
```

---

### 3. メール・チャット履歴

**優先度**: ★★★☆☆（推奨）

**理由**:
- 「素」の言葉が表れる
- 判断軸が明確（仕事のメールなど）
- 関係性が表れる

**収集方法**:

#### Gmail

```
1. Gmailを開く
2. 検索ボックスに「from:me after:2025/11/01」と入力
3. 自分が送信したメールを抽出
4. 重要なメールをコピー
```

#### Slack

```
1. Slackを開く
2. 検索ボックスに「from:me after:2025-11-01」と入力
3. 自分が送信したメッセージを抽出
4. 重要なメッセージをコピー
```

---

### 4. 日記・メモ

**優先度**: ★★★☆☆（推奨）

**理由**:
- 個人的な思考が表れる
- 感情ログとして活用できる
- 価値観が明確

**収集方法**:

```
1. 日記アプリ、メモアプリを開く
2. 過去3-6ヶ月分の記録をコピー
3. 個人的すぎる内容は除外してもOK
```

---

### 5. プレゼン資料・レポート

**優先度**: ★★☆☆☆（任意）

**理由**:
- 専門分野の知識が表れる
- 論理的な思考プロセスが明確

**収集方法**:

```
1. 過去1年分のプレゼン資料、レポートをリストアップ
2. テキスト部分を抽出
3. 1つのファイルにまとめる
```

---

### 6. 動画・音声の文字起こし

**優先度**: ★★☆☆☆（任意）

**理由**:
- 話し言葉が表れる
- 専門分野の知識が表れる

**収集方法**:

```
1. YouTube動画、ポッドキャストのURLをリストアップ
2. 文字起こしツール（Otter.ai, Whisperなど）で文字起こし
3. テキストをコピー
```

## 📊 データの整理

### ファイル形式

```
推奨: Markdown形式（.md）
理由: 構造化しやすく、AIが読みやすい
```

### ファイル構成例

```
collected_data/
├── sns_posts.md           # SNS投稿
├── blog_articles.md       # ブログ記事
├── emails_chats.md        # メール・チャット
├── diary_notes.md         # 日記・メモ
├── presentations.md       # プレゼン資料
└── transcripts.md         # 動画・音声の文字起こし
```

### ファイル内容の例

#### sns_posts.md

```markdown
# SNS投稿（2025年11月〜2026年2月）

## 2026-02-11

[投稿内容]

いいね: 350
リプライ: 45

---

## 2026-02-10

[投稿内容]

いいね: 120
リプライ: 10

---

（以下、同様に記録）
```

## 🎯 データ収集の目標

### 最低限の目標

```
□ SNS投稿: 100件以上（過去3ヶ月分）
□ ブログ記事: 5-10記事（過去1年分）
□ メール・チャット: 50件以上（過去3ヶ月分）
```

### 理想的な目標

```
□ SNS投稿: 200件以上（過去6ヶ月分）
□ ブログ記事: 10-20記事（過去1年分）
□ メール・チャット: 100件以上（過去6ヶ月分）
□ 日記・メモ: 30件以上（過去3ヶ月分）
```

---

## 🔍 データの質を高めるコツ

### 1. 多様性を確保する

```
□ 異なる媒体から収集（SNS, ブログ, メールなど）
□ 異なるトピックを含める（仕事, 趣味, 日常など）
□ 異なる感情状態の投稿を含める（興奮, 落ち着き, 怒りなど）
```

### 2. 最近のデータを優先する

```
□ 過去3-6ヶ月のデータを中心に収集
□ 古すぎるデータ（1年以上前）は、価値観が変わっている可能性がある
□ 最新の言語スタイルを反映させる
```

### 3. 「素」の言葉を優先する

```
□ 公式な文章よりも、カジュアルな投稿を優先
□ 「本音」が表れている投稿を優先
□ テンプレート的な投稿は除外
```

---

## 🚫 避けるべきデータ

### 収集しないほうがいいデータ

```
□ 他者のプライバシーに関わる内容
□ 機密情報・社外秘の内容
□ 古すぎるデータ（2年以上前）
□ テンプレート的な投稿（「おはようございます」など）
□ 他者の引用だけの投稿
```

---

## 📝 データ収集のチェックリスト

### 収集完了の確認

```
□ SNS投稿を100件以上収集した
□ ブログ記事を5件以上収集した
□ メール・チャットを50件以上収集した
□ データをMarkdown形式で整理した
□ 個人情報・機密情報を除外した
□ 最近3-6ヶ月のデータを中心に収集した
```
