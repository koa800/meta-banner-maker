# 抽出テンプレート

> **目的**: 収集したデータから、価値観・発想パターン・判断軸・言語スタイルを抽出する方法を説明する

---

## 📋 抽出の流れ

```
ステップ1: データを読み込む
    ↓
ステップ2: AIに分析を依頼する
    ↓
ステップ3: 抽出結果を確認・修正
    ↓
ステップ4: 各ファイルに記入
```

---

## 🤖 AIを使った抽出（推奨）

### ステップ1: データを読み込む

```
1. 収集したデータ（sns_posts.md, blog_articles.mdなど）を用意
2. AIに読み込ませる
```

---

### ステップ2: 抽出プロンプトを実行

以下のプロンプトをコピーして、AIに送信してください。

---

## 📝 抽出プロンプト1: 価値観（審美眼）

```
【依頼】
以下のデータから、私の価値観を抽出して：

■ データ
[ここに、sns_posts.md, blog_articles.mdなどの内容を貼り付け]

■ 抽出してほしい情報
1. 最上位の価値観（優先順位順に5つ）
   - 各価値観について、なぜ大切にしているか理由を説明
   - 各価値観が表れた具体例を挙げる

2. 好きなもの・惹かれるもの
   - カテゴリ別（思考・アイデア、行動・姿勢、人・関係性、コンテンツ・表現）
   - 各項目について、なぜ好きか理由を説明
   - 具体例を挙げる

3. 嫌いなもの・避けたいもの
   - カテゴリ別（思考・アイデア、行動・姿勢、人・関係性、コンテンツ・表現）
   - 各項目について、なぜ嫌いか理由を説明
   - 具体例を挙げる

■ 出力フォーマット
1_Core/SOUL.mdの「最上位の価値観」「好きなもの」「嫌いなもの」のセクションに、そのまま貼り付けられる形式で出力してください。
```

## ✍️ ステップ3: 抽出結果を確認・修正

### 確認ポイント

```
□ 抽出結果は、自分の感覚と一致しているか？
□ 具体例は、実際の発言・行動と一致しているか？
□ 不足している情報はないか？
□ 誤解を招く表現はないか？
```

### 修正のコツ

```
1. AIの抽出結果をそのまま使わない
   - 自分の言葉で書き直す
   - 具体例を追加する

2. 優先順位を明確にする
   - 最も重要な価値観を1位にする
   - よく使うパターンを上位にする

3. 具体例を増やす
   - 抽象的な説明だけでなく、具体例を多く入れる
   - AIは具体例から学習する
```

## 🔄 手動で抽出する場合

AIを使わずに、手動で抽出することもできます。

### 手動抽出の流れ

```
1. 収集したデータを読む
   ↓
2. 以下の質問に答える形で、メモを取る
   ↓
3. メモを整理し、各ファイルに記入
```

### 質問リスト

#### 価値観

```
□ 私が最も大切にしていることは何か？（5つ）
□ 私が好きなものは何か？（思考、行動、人、コンテンツ）
□ 私が嫌いなものは何か？（思考、行動、人、コンテンツ）
□ 過去の投稿で、繰り返し語っているテーマは何か？
```

#### 発想パターン

```
□ 私はどのように考えるか？
□ アイデアを出す時、どんなパターンを使っているか？
□ 他者と違う視点を持つ時、どんな思考をしているか？
□ 過去のアイデアで、うまくいったものはどんなパターンで生まれたか？
```

#### 判断軸

```
□ 私は何を優先して判断しているか？
□ トレードオフの場面で、どちらを選ぶ傾向があるか？
□ 過去の重要な判断で、何を基準に決めたか？
```

#### 言語スタイル

```
□ 私はどんなトーンで話すか？
□ よく使う言葉・フレーズは何か？
□ 避けている言葉・フレーズは何か？
□ 文の長さは？（短文が多い？長文が多い？）
□ どんな表現技法を使っているか？（比喩、対比、問いかけなど）
```
