# company レイヤー

最終更新: 2026-03-08

`Master/company/` は、会社共通で参照してよい事実の正規レイヤーです。4層で見ると `前提` と `knowledge` の中間にあり、特に「公開してよい人物理解」を持ちます。

## 現在の正本

- `people_public.json`
  会社共通で見てよい人物情報の正規レイヤー

## ここに置くもの

- 役割
- 表面的プロフィール
- 共有してよい業務情報
- 検証済みの事実

## ここに置かないもの

- 深い人物理解
- 未検証の仮説
- 甲原だけが持つべき private な見立て

private な人物理解は `Master/brains/*/people_private.json` に寄せる。
