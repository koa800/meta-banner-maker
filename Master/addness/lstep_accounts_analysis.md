## 1. アカウント全体像

### 稼働中 / 停止中の判定ルール

- Lステップのアカウントは、`稼働中アカウント一覧` と `利用停止中アカウント一覧` を分けて扱う
- `導線データ / Lステップ一覧` シートには、`稼働状況` と `利用停止日` を追加し、停止中アカウントも同じ一覧で管理する
- 導線の接続先として有効とみなすのは、原則 `稼働中アカウント一覧` にあるものだけ
- `利用停止中アカウント一覧` に載っているIDは、既存導線に残っていても `旧アカウントへの誤接続候補` として扱う
- 2026-03-08 時点で `@804mrsmd` は `利用停止中アカウント一覧` にあり、AI / SNS の個別3日間導線一式に残っているのは誤接続

#### 代表例

   - `https://school.addness.co.jp/page/KAtTy0tucas2`
   - `https://school.addness.co.jp/page/vQ8ZuYjSpxuh`

   - `https://school.addness.co.jp/page/kjOhv6Z6lU8r`
   - `https://school.addness.co.jp/page/bbwZzf9MxRUb`
5. その UTAGE ページを開き、ページソースか開発者ツール内で `804mrsmd` を検索する

8. これで `今送っているメール -> 今のUTAGEページ -> 停止中LINE` がつながっていると判断できる

#### AI 側の代表例

AI なら `UTAGE_AIカレッジ_Facebook_個別3日間シナリオ 2025-10-15` を見る。

## 4. 導線フロー全体まとめ

### みかみアカウント（AI/SNSファネル）

```
広告(Meta/X/TikTok/YouTube) → LP → メール取得 → OTO(¥2,980)
  → みかみ公式LINE登録（流入経路タグで媒体×バリアント特定）
    → プロコンサル or 本気コンサルシナリオ（媒体別に自動分岐）
      → ファン化動画_視聴シナリオ①→② （教育コンテンツ視聴）
        → 感想記入シナリオ
          → エンドアクション → エバー導線
            → 個別相談（zoom）→ 成約
```

### スキルプラスアカウント

```
広告(Meta/アフィリエイト) → LP → メール取得
  → スキルプラス公式LINE登録（流入経路タグ付与）
    → セミナー導線_予約シナリオ①→②→③ （セミナー予約促進）
      → 「プロモ対象」タグ付け → フリープランシナリオ
        → スキル習得セミナー参加
          → ロードマップ作成会オファー
            → 個別相談 → 成約
```

## 6. 用語・概念の整理

| フリープランシナリオ | スキルプラスのフリープラン案内シナリオ |
