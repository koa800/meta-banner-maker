# Meta広告「大当たり」CR分析レポート

最終更新: 2026-03-03

## 分析概要

- 対象期間: 2023/01/01 - 2026/03/02
- 対象: CR判定「大当たり」全340件
- データソース: Looker Studio CR一覧ダッシュボード（CSVエクスポート）
- 合計消化金額: 約8.0億円 / 合計インプレッション: 約1.5億
- 合計集客数: 98,028人（データあり分）
- 生データ: `System/data/cr_bigwinners_raw.csv`

### 用語定義

- **CPA**: 集客単価（広告費 / 集客数）。メールアドレス取得 or LINE登録1件あたりのコスト
- **CPO**: フロント購入単価（広告費 / フロント購入数）。メール取得直後のワンタイムオファー（¥2,980フロント商品）の購入1件あたりのコスト。CPOが低い = フロント購入率が高い = 質の高い集客ができている指標

**キーワード**: 「with」「堀江」「林社長」「ヒカル」「箕輪」

**⚠️ 重要な落とし穴（甲原の知見）:**
著名人CRは短期的にCPA・CPOが大幅に改善するが、**中長期では逆効果になる**。

- **市場の拡大問題**: 著名人に興味がある層が反応するため、本来ターゲットにしていない市場まで広がってしまう
- **広告学習の最適化崩壊**: Metaの配信学習が著名人興味層に寄ってしまい、自社ブランド単体のCRの配信効率が悪化する
- **依存リスク**: 著名人CRに頼ると、その人なしでは獲得できない構造になる。ドラッグのようなもの
- **本質的でない集客**: 著名人だから反応した人は、商品・サービスそのものへの関心が薄い

**結論**: 数値上は50本の大当たりがあるが、**権威訴求を軸にしたCR企画は推奨しない**。他パターン（A〜E）の自社ブランド訴求を主軸とし、権威はあくまで補助的な要素に留めるべき。

## 5. 大当たりCRの7つの共通点（340件から再検証）

### 1. 感情を動かすフック（340本に共通）
「足りてない」「後悔」「やめました」「おい」「二極化」「急募」「間違い」など、**スクロールを止める強い言葉**が広告名に含まれる。中性的なタイトルのCRは極めて少ない。

### 2. 動画フォーマット優位（310/340本 = 91%）
大当たりの9割以上が動画。画像CRは「急募」「ASC」等の限定的パターンのみ。

### 3. 痛み > 利得
「足りてない」「後悔」「二極化」「終了」「間違い」など、**行動しないリスク**を訴求するCRが大多数。損失回避バイアスの活用。

### 4. ターゲット明示
「会社員」「フリーランス」「副業」「50代」「デザイナー」など、**誰に向けた広告か**が明確。

### 5. フック率25%以上が基本ライン
大当たり動画の中央値28.5%。ただし二極化系（17-18%）のように低フック率でもCVRで挽回するケースもある。

### 6. 「型」の横展開が最も効率的
同テーマの派生CRが大量に大当たり:
- 「足りてない」系 → AI/SNS合わせて30本以上
- 「全部やめました」系 → 16本
- 「二極化」系 → 16本
- 「問題ないです」系 → 15本以上
- 「AI学ぶな」系 → 8本

### 7. 権威者CRは「ドラッグ」— 短期効果に騙されない
「with堀江」「with林社長」「ヒカルさん」の冠がついたCRが50本あり、CPA・CPOの数値は良い。しかし**著名人CRは中長期で広告学習を崩壊させる罠**。著名人の興味層にMeta配信が寄り、自社ブランドCRの効率が悪化する。数値だけ見ると「当たり」に見えるが、アカウント全体のROASを下げる原因になりうる。

## 6. パターン別の「使い方」（CR企画スキル連携）

### CPA最安を狙うなら → D.実用訴求
- 「具体的に何が学べるか」を明示 → CPA ¥2,100-2,800台
- 「プロンプト_お絵描きムービー」「canvaデザイン術」のように具体的
- CPOも最安（中央値¥30,584）→ 後ろの数値も良い

### 大量配信を狙うなら → A.危機感訴求
- 最も安定して消化が伸びる王道パターン（総消化¥2.64億）
- 「1年後悔」は¥3,277万消化。枯れにくい
- 「これ間違いです」はフック率66%の新型。横展開の余地大

### CPOを下げたいなら → D.実用訴求を深掘り
- 「プロンプト_お絵描きムービー」CPO¥20,960 / 「canvaデザイン術」CPA¥1,215
- 「何が学べるか」を具体的に示す → 期待とLPが一致 → 後ろの数値も良い
- **⚠️ 権威者との掛け合わせ（F.権威）は短期的にCPOが下がるが、広告学習を歪めるため非推奨**

### 新ファネルを開拓するなら
- **スキルプラス**: 7件の大当たりあり。「おい会社員(演出あり)デザイン強化」等
- **デザイン**: CPA¥1,215（全CR最安）を記録。canva/Coconala訴求が有効

### 注意: 質の低い集客を避けるには
- **F.権威訴求は短期的にCPA・CPOが下がるが、広告学習を歪めて中長期でROASを悪化させる。主軸にしない**
- E.急募/FOMO訴求はCPO中央値¥38,488と高い → 集客は安いが後ろが悪い傾向
- C.二極化訴求はCPA高め（¥4,467）だが後ろは悪くない
- **CPAだけで判断しない。CPO・ROASとセットで評価する**

## 7. 時系列トレンド

### 2023年（24件）— 黎明期
- AIファネルの「足りてない」「急募」が主力
- 製作者データほぼなし（属人化していた時代）
- 年間消化: 約1.04億円

### 2024年（94件）— 拡大期
- 在中悠也のプロンプト系・二極化系が大当たり連発
- SNSファネルが40件に成長
- 「問題ないです」シリーズ登場
- 年間消化: 約2.88億円（前年比2.8倍）

### 2025年（193件）— 爆発期
- 大当たり件数が倍増（94→193）
- 権威訴求（with堀江・林社長）が増加したが、中長期では広告学習を歪める落とし穴が判明
- 「これ間違いです」系がフック率66%の新記録
- デザイン・スキルプラスファネルが拡大
- 鈴木織大が製作者として44件の量産体制を確立
- 年間消化: 約3.76億円（前年比1.3倍）

### 2026年（29件、3月時点）
- 1年後悔系の新バージョンが引き続き好調
- スキルプラスファネルにも大当たりが出始めている
