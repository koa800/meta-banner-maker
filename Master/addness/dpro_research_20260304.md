# 広告リサーチレポート 2026-03-04

> **リサーチ手法**: DPro（Instagram × 3キーワード） + Meta広告ライブラリ（AI 副業） + Instagram SNS（#ai副業） + 自社商品分析
> **分析方針**: 大手企業を除外、中小企業に絞り、消化額マトリクスで分類。長期配信 = 当たっている広告として注目
> **インサイト分析の軸**: 各参考元CRが触れている「避けたいコスト × 満たしたい需要のバランス」を読み取る

## CR企画の前提条件（2026-03-06 確定）

| 項目 | 内容 |
|------|------|
| **対象ファネル** | AIファネル（行き先LINE: 【みかみ】アドネス株式会社） |
| **目的** | 集客数の回復（既存CRの疲弊対策） |
| **現在の集客層** | 「AIで稼げる」「AI副業」意識が強い層 → LTVにギャップあり |
| **理想ターゲット** | 事業者・経営者層（LTV高い） |

### 理想ターゲットの3条件

| 条件 | 心理状態 |
|------|---------|
| **(a) AI出力への不満** | AIの出力が物足りず「自分でやった方が早い」と思っている |
| **(b) 時間への危機感** | このままでは自分の時間が削られ続け、膨大な時間がかかることに危機感がある |
| **(c) 教えてくれる人が欲しい** | 効率化のために、誰かにAIの高度な使い方を教えてほしいと考えている |

### 現状とのギャップ

現在のAIファネルCR（大当たり158件）の主要訴求:
- 「おい会社員」「急募訴求」「二極化（フリーランス/副業会社員）」「AI足りてない」「プロンプト」「貯金」
- → **「AI副業で稼ぐ」「会社員に危機感を煽る」系が大半**
- → 理想ターゲット（事業者・経営者層）の心理（a)(b)(c)とは異なる訴求

**今回のリサーチも「AI 副業」「#ai副業」キーワードで実施したため、副業系のCRが多く出ている。** 理想ターゲット向けには「AI 業務効率化」「AI DX」「ChatGPT ビジネス活用」等のキーワードでの追加リサーチが必要。

### CDPデータからの示唆

1. **甲原さんの仮説はデータで裏付けられた**: 現在の顧客の約61%が「マネタイズ」（副業で稼ぐ）志向。理想ターゲット（ビジネスにAI導入で効率化）は約15%
2. **経営者・役員の平均LTVは会社員の2.7倍**: ¥609,269 vs ¥224,992。事業者・経営者層を増やせばLTVが大幅に上がる
3. **AIファネルは最大の高LTV顧客供給源**: 高LTV顧客（¥50万以上）の55%がAIファネル経由
4. **理想ターゲットは「すでに来ている」**: 1,421人が「ビジネスにAI導入」を選択。うち33%が事業者・経営者。CRが彼ら向けではないのに来ている → 訴求を合わせれば増やせる
5. **悩みセグメント単体ではLTV差は小さい**: 理想ターゲット ¥105K vs 副業マネタイズ ¥99K（1.06倍）。**LTVの最大のドライバーは「悩み」ではなく「職業」**（経営者・役員: ¥609K = 会社員の2.7倍）
6. **つまり「事業者・経営者層の流入比率を上げる」ことが最もLTVインパクトが大きい**: 悩みの内容を変えることより、来る人の属性（職業）を変えることが重要。CRの訴求・世界観・使う言葉で「事業者・経営者が反応するCR」を作ることが鍵

### 参考元 1: 成島拓｜コンテンツビジネスのプロ

**実際のURL:**
- LP/ブログ: https://ntk171.com/
- DPro上: Instagram「副業」キーワード4位

**消化額マトリクス**: 非常に優秀（累計¥2,674,648 × 直近¥215,622）

**インサイト分析（避けたいコスト × 満たしたい需要）:**

| 避けたいコスト | 満たしたい需要 |
|-------------|-------------|
| **顔出し（身バレ）** — 「"顔出しなし"で情報発信」を繰り返し強調 | コンテンツビジネスで副収入を得る |
| **会社を辞めること** — 「会社員と副業を両立」= 今の安定を失いたくない | 自分の商品を持つ（「売れる商品を創るノウハウ」） |
| **複雑なマーケティング習得** — プロが教えてくれる = 自力の試行錯誤を避けたい | 凡人でも勝てる感覚（「凡人のための負けない人生戦略」） |

**最適解のインサイト:** 「顔出しせず、会社を辞めず、でもコンテンツビジネスで本気で稼げる」

**なぜ累計¥267万も配信し続けているか:** このインサイトが普遍的に刺さるから。「今の立場を守ったまま新しい収入を得たい」という矛盾を、「ハイブリッド起業戦略」というフレームで解決している。

**アドネスへの転用:**
- アドネスのターゲットも同じ矛盾を持つ: 「今の仕事のリスクを取りたくない」×「AIスキルで新しい収入を得たい」
- 「顔出しなし」の代わりに「今の仕事を続けながら」「スキマ時間で」のような表現で同じ構造を作れる
- ただし「スキマ時間で」は「楽に」に近づくリスクあり。「本業の合間に」「週末だけで」のように具体性を持たせる
