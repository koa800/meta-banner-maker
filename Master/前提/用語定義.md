## 広告媒体別参照ルール

このセクションは、広告媒体ごとに指標定義がずれる時に、AI がどの公式一次ソースを正本として参照するかを固定するためのもの。

### Meta広告

- 優先参照
  - `https://developers.facebook.com/docs/marketing-api/reference/ad-account/insights/`
  - `https://developers.facebook.com/docs/marketing-api/insights/`
  - `https://developers.facebook.com/docs/marketing-api/insights/estimated-in-development`
  - `https://developers.facebook.com/ads/blog/post/v2/2016/03/16/link-clicks-updates/`
- 参照ルール
  - `クリック` 系が曖昧な時は `clicks` ではなく、まず `inline_link_clicks` / `link_click` を確認する。
  - `reach` と `frequency` は推定値として扱い、他媒体の確定値と同列比較しない。

### X広告

- 優先参照
  - `https://business.x.com/en/help/overview/twitter-ads-glossary`
  - `https://business.x.com/en/help/campaign-measurement-and-analytics/campaign-dashboard`
  - `https://business.x.com/en/help/campaign-measurement-and-analytics/tweet-activity-dashboard`
- 参照ルール
  - `CTR` と `engagement rate` は別物として扱う。
  - `results` は目的依存なので、目的が分からない時は `link clicks` や `engagements` に勝手に読み替えない。

### TikTok広告

- 優先参照
  - `https://ads.tiktok.com/help/article/basic-data?lang=en`
  - `https://ads.tiktok.com/help/article/all-metrics?lang=en`
  - `https://ads.tiktok.com/help/article/video-play?lang=en`
  - `https://ads.tiktok.com/help/article/onsite-events?lang=en`
- 参照ルール
  - `clicks (all)` と `clicks (destination)` を混ぜない。
  - `CVR` は `impressions 基準` と `destination clicks 基準` があるため、母数を必ず固定する。

### LINE広告

- 優先参照
  - `https://www.lycbiz.com/jp/column/line-ads/technique/20210728/`
  - `https://www.lycbiz.com/jp/manual/line-ads/other_007/`
  - `https://www.lycbiz.com/sites/default/files/media/jp/ebook/line-ads/support/operational-type-ads-basic-knowledge.pdf`
- 参照ルール
  - Web CV と LINE友だち追加は別成果として扱い、`CPA` と `CPF` を混ぜない。
  - オークション分析の `インプレッションシェア` と通常の `インプレッション` を混ぜない。

### Google広告

- 優先参照
  - `https://developers.google.com/google-ads/api/fields/v23/metrics`
  - `https://support.google.com/google-ads/answer/2615875?hl=en`
  - `https://support.google.com/google-ads/answer/6310?hl=en`
  - `https://support.google.com/google-ads/answer/6270625?hl=en`
  - `https://support.google.com/google-ads/answer/2375431?hl=en`
- 参照ルール
  - API の `cost_micros` は表示金額に直すまで `金額` とみなさない。
  - `Google広告` の動画系指標は `YouTube 広告` を含むため、 view 系は YouTube の定義を優先する。

### LinkedIn広告

- 優先参照
  - `https://www.linkedin.com/help/lms/answer/a445476`
  - `https://www.linkedin.com/help/lms/answer/a422642`
  - `https://www.linkedin.com/help/lms/answer/a422348`
- 参照ルール
  - `Clicks` は `chargeable clicks` を基準にする。
  - `Key Results` と `Cost per Result` は objective 依存なので、目的とセットで扱う。

### Pinterest広告

- 優先参照
  - `https://help.pinterest.com/en/business/article/reporting-dashboard-overview`
  - `https://help.pinterest.com/en-gb/business/article/media-planner`
- 参照ルール
  - `Pin clicks` と `Outbound clicks` を分ける。
  - `Weekly frequency` は通常 frequency と別指標として扱う。

### Snapchat広告

- 優先参照
  - `https://forbusiness.snapchat.com/blog/remarketing-how-it-works`
  - 目的別の公式ページを追加取得できたら、blog よりそちらを優先する。
- 参照ルール
  - 現時点では structured glossary が弱いため、指標を他媒体の定義で補完しない。
  - `CTR / conversion rate / CPC / ROAS / frequency` は Snap 公式文脈で確認したものだけ使う。

## 広告媒体別の主要指標定義

### Meta広告

| 用語 | 定義 | 補足 |
|---|---|---|
| Meta広告 / impressions | 広告が画面に表示された回数。 | `ad-account/insights` 基準。 |
| Meta広告 / reach | 広告を少なくとも1回見たアカウント数。 | 推定値。 |
| Meta広告 / frequency | 1人あたりの平均表示回数。 | 推定値。 |
| Meta広告 / clicks | 広告上の総クリック数。 | ページいいね等も含みうる。 |
| Meta広告 / inline_link_clicks | Meta 内外のリンク先や体験へのリンククリック数。 | `clicks` より link click 解釈を優先。 |
| Meta広告 / spend | 推定総消化金額。 | `ad-account/insights` 基準。 |
| Meta広告 / CPC | クリック単価。 | Meta では link click を中心に解釈する。 |
| Meta広告 / CPM | 1,000インプレッションあたりの平均コスト。 | `ad-account/insights` 基準。 |
| Meta広告 / CTR | クリック数をインプレッション数で割った率。 | `clicks / impressions`。 |

### X広告

| 用語 | 定義 | 補足 |
|---|---|---|
| X広告 / impressions | X 上で広告が表示された回数。 | campaign dashboard 基準。 |
| X広告 / results | キャンペーン目的に直接ひもづく成果数。 | 目的依存。 |
| X広告 / clicks | キャンペーンや構成要素への直接クリック数。 | glossary 基準。 |
| X広告 / link clicks | 投稿内 URL へのクリック数。 | PAD 基準。 |
| X広告 / engagements | 投稿への総インタラクション数。 | いいね、返信、リンククリック等を含む。 |
| X広告 / engagement rate | エンゲージメント数をインプレッション数で割った率。 | PAD 基準。 |
| X広告 / CTR | 総エンゲージメント数をインプレッション数で割った率。 | glossary 基準。 |
| X広告 / CPC | 総支出をクリック数で割った値。 | glossary 基準。 |
| X広告 / CPE | 総支出をエンゲージメント数で割った値。 | glossary 基準。 |
| X広告 / cost per result | 総支出を relevant action 数で割った値。 | campaign dashboard 基準。 |

### TikTok広告

| 用語 | 定義 | 補足 |
|---|---|---|
| TikTok広告 / impressions | 広告が表示された回数。 | `basic-data` 基準。 |
| TikTok広告 / clicks (all) | social interactions を含む広告クリック数。 | `clicks (destination)` と別。 |
| TikTok広告 / clicks (destination) | 指定した遷移先へ進んだクリック数。 | 遷移系の正本。 |
| TikTok広告 / reach | 広告を1回以上見たユニークユーザー数。 | `basic-data` 基準。 |
| TikTok広告 / frequency | 一定期間で各ユーザーが広告を見た平均回数。 | `basic-data` 基準。 |
| TikTok広告 / conversions | 選択した optimization event が起きた回数。 | `basic-data` 基準。 |
| TikTok広告 / CVR | コンバージョン数をインプレッション数で割った率。 | `CVR, clicks` と別。 |
| TikTok広告 / CVR, clicks | コンバージョン数を destination clicks で割った率。 | クリック基準の CVR。 |
| TikTok広告 / cost | 総広告費。 | `basic-data` 基準。 |
| TikTok広告 / CPC (destination) | destination click 1回あたりの平均コスト。 | `basic-data` 基準。 |
| TikTok広告 / cost per conversion | コンバージョン1件あたりの平均コスト。 | `basic-data` 基準。 |
| TikTok広告 / video views | 動画が再生開始された回数。 | `video-play` 基準。 |
| TikTok広告 / 2-second video views | 2秒以上再生された回数。 | `video-play` 基準。 |
| TikTok広告 / 6-second video views | 6秒以上再生、または全編再生や初期 engagement を満たした回数。 | `video-play` 基準。 |
| TikTok広告 / video views at 100% | 動画が最後まで再生された回数。 | `video-play` 基準。 |

### LINE広告

| 用語 | 定義 | 補足 |
|---|---|---|
| LINE広告 / impressions | 広告が表示された回数。 | PDF 用語集基準。 |
| LINE広告 / CTR | クリック数をインプレッション数で割った率。 | 公式コラム基準。 |
| LINE広告 / CPC | 1クリックあたりのコスト。 | 公式コラム基準。 |
| LINE広告 / CPM | 1,000回表示あたりのコスト、または課金方式。 | 公式コラム基準。 |
| LINE広告 / CV | 広告クリック後に起きた成果行動。 | 商品購入、資料DL、問い合わせ等。 |
| LINE広告 / CVR | CV 数をクリック数で割った率。 | 公式コラム基準。 |
| LINE広告 / CPA | 1件の成果獲得にかかった単価。 | Web CV 系。 |
| LINE広告 / CPF | LINE 友だち追加1件あたりの単価。 | LINE 独自の友だち追加系。 |
| LINE広告 / インプレッションシェア | 広告表示機会総数に対して実際に表示された割合。 | オークション分析用。 |
| LINE広告 / インプレッション損失率（予算） | 予算不足で表示されなかった割合。 | オークション分析用。 |
| LINE広告 / インプレッション損失率（ランク） | 広告ランクが低く負けたことで表示されなかった割合。 | オークション分析用。 |
| LINE広告 / ランク | 入札額と広告の予測値から構成される配信候補ランキング。 | オークション分析用。 |
