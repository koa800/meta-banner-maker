# output レイヤー

最終更新: 2026-03-08

ここは、実際に出した成果物と、その結果を残す場所です。大事なのは「出したもの」より「次回の判断に戻せる形で残すこと」です。

## 置くもの

- LP案
- コピー案
- メッセージ案
- 企画案
- リサーチアウトプット
- 経理実績に紐づく領収書・請求書
- それぞれの結果レビュー

## 置かないもの

- まだ出していない思考メモだけの草案
- 汎用原理だけを抜き出した知識
- 禁止事項だけのルール集

## 残し方

- 成果物そのもの
- 何のために作ったか
- 誰に向けたか
- どんな仮説で作ったか
- 結果がどうだったか
- knowledge に戻す学び
- rules に戻す禁止・推奨

テンプレートは `Master/output/OUTPUT_REVIEW_TEMPLATE.md` を使う。

## 命名の基本

- `YYYY-MM-DD_件名.md` で残す
- 既存資産を整理しただけのものも、再利用判断に効くならここへ残す
